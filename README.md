# CORTHEX HQ

**AI Agent Corporation** — 29명의 AI 에이전트가 회사처럼 운영되는 멀티 에이전트 자동화 시스템

> 대표(사용자)가 한국어로 명령을 내리면, 비서실장이 판단해서 적절한 부서에 배분하고,
> 전문가들이 병렬로 작업한 뒤, 결과를 종합 보고서로 합쳐서 돌려줍니다.

**라이브**: https://corthex-hq.com

---

## 한 줄 요약

**"법학 전공자가 AI와 대화만으로 29명의 AI 직원이 일하는 회사를 만들었다"**

## 핵심 특징

| 특징 | 설명 |
|------|------|
| **29명 AI 에이전트** | 비서실장 -> 처장(Manager) -> 전문가(Specialist) 실제 회사 계층 |
| **자동 업무 배분** | "삼성전자 분석해줘" -> 비서실장이 CIO에게 배분 -> 4명 전문가가 병렬 분석 -> 종합 보고 |
| **자율 딥워크** | 에이전트가 계획->조사->초안->정리->보고까지 자율적으로 수행 |
| **Glass + Aurora 디자인** | 유리 반투명 효과 + 오로라 그라데이션의 현대적 대시보드 |
| **멀티 LLM** | Claude 4.5/4.6 + GPT-5/5.2 + Gemini 2.5/3.1 — 10종 모델, 에이전트별 개별 설정 |
| **도구 125개+** | 투자분석, 법률검토, 마케팅, SNS 발행 등 전문 도구 |
| **실시간 자동매매** | KIS 연동, 한국/미국 주식 실매매 가능 |
| **텔레그램 지휘** | 텔레그램으로 어디서든 AI 에이전트에게 명령 |
| **24시간 무료 서버** | Oracle Cloud ARM 4코어 24GB, 월 $0 |
| **완전 자동 배포** | `[완료]` 커밋 -> 자동 PR -> 자동 머지 -> 서버 배포 |

## 기술 스택

| 구성 요소 | 기술 |
|-----------|------|
| **백엔드** | Python 3.11+ / FastAPI |
| **프론트엔드** | Tailwind CSS + Alpine.js (CDN) |
| **디자인** | Glass + Aurora Mix / hq-* CSS 토큰 194개+ |
| **AI 모델** | Claude 4.5/4.6, GPT-5/5.2, Gemini 2.5/3.1 (10종) |
| **DB** | SQLite (서버 로컬, 배포 독립) |
| **서버** | Oracle Cloud ARM (무료) + nginx + HTTPS |
| **배포** | GitHub Actions 자동 (커밋 -> PR -> 머지 -> 배포) |
| **도메인** | corthex-hq.com (Cloudflare) |
