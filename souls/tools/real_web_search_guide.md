# real_web_search — 실시간 웹 검색 도구 가이드

## 이 도구는 뭔가요?
SerpAPI를 통해 **실제 구글 검색**을 수행하는 도구입니다.
web_search(AI 지식 기반)와 달리 진짜 인터넷에서 검색하므로 **최신 정보**를 가져올 수 있습니다.
웹 검색, 뉴스 검색, 이미지 검색 3가지 기능을 지원합니다.

## 어떤 API를 쓰나요?
- **SerpAPI** (https://serpapi.com) — 구글 검색 결과를 API로 제공하는 서비스
- 비용: **유료** (월 100회 무료, 이후 유료 플랜)
- 필요한 키: `SERPAPI_KEY`

## 사용법

### action=search (웹 검색)
```
action=search, query="검색어", num=10
```
- `query` (필수): 구글에서 검색할 키워드
- `num` (선택, 기본 10): 검색 결과 개수

반환 정보: 제목, 링크, 요약(snippet) + AI 종합 분석

**예시:**
- `action=search, query="LEET 2026 일정"` → 최신 LEET 시험 일정 검색
- `action=search, query="법학적성시험 시장 규모 2025"` → 시장 규모 최신 정보

### action=news (뉴스 검색)
```
action=news, query="검색어", num=10
```
- `query` (필수): 뉴스 검색 키워드
- `num` (선택, 기본 10): 뉴스 결과 개수

반환 정보: 뉴스 제목, 출처, 날짜, 링크, 내용 요약 + AI 트렌드 분석

**예시:**
- `action=news, query="로스쿨 정원 확대"` → 로스쿨 관련 최신 뉴스
- `action=news, query="LEET 해설 논란"` → LEET 해설 관련 뉴스

### action=image (이미지 검색)
```
action=image, query="검색어", num=5
```
- `query` (필수): 이미지 검색 키워드
- `num` (선택, 기본 5): 이미지 결과 개수

반환 정보: 이미지 제목, URL, 출처

**예시:**
- `action=image, query="LEET Master 로고"` → 관련 이미지 검색

## 이 도구를 쓰는 에이전트들

### 1. 비서실장 (Chief of Staff)
**언제 쓰나?** CEO의 질문에 최신 정보가 필요할 때
**어떻게 쓰나?**
- 최신 뉴스, 시장 동향, 경쟁사 소식 등 실시간 정보 수집

### 2. 정보 보좌관 (Information Aide)
**언제 쓰나?** 보고서 작성 시 최신 데이터 확인
**어떻게 쓰나?**
- 뉴스 검색으로 최신 업계 동향 파악
- 수치 데이터 교차 검증

### 3. 검수 보좌관 (Relay Specialist)
**언제 쓰나?** 보고서 내용의 최신성 검증
**어떻게 쓰나?**
- 보고서에 인용된 정보가 여전히 유효한지 실시간 검색으로 확인

### 4. 사업기획처장 (CSO)
**언제 쓰나?** 사업 기획 시 최신 시장 정보 수집
**어떻게 쓰나?**
- 경쟁사 최신 동향, 규제 변화, 시장 트렌드 파악

### 5. 시장조사 Specialist
**언제 쓰나?** 시장 조사 시 최신 데이터 수집
**어떻게 쓰나?**
- naver_datalab(검색량)과 함께 시장 동향 종합 분석
- 뉴스 검색으로 업계 이슈 파악

### 6. 법무·IP처장 (CLO)
**언제 쓰나?** 최신 법률/판례/규제 동향 확인
**어떻게 쓰나?**
- 최신 법률 개정 소식, 판례 동향 실시간 검색

### 7. 설문/리서치 Specialist
**언제 쓰나?** 리서치 시 최신 업계 벤치마크 확인
**어떻게 쓰나?**
- 설문 주제 관련 최신 통계/보고서 검색

### 8. 투자분석처장 (CIO)
**언제 쓰나?** 투자 판단 시 최신 시장 정보 수집
**어떻게 쓰나?**
- 금융 시장 최신 뉴스, 기업 소식 실시간 검색
- ecos_macro(경제지표), kr_stock(주가) 등과 함께 종합 분석

### 9. 시황분석 Specialist
**언제 쓰나?** 거시경제/시장 동향 최신 정보 확인
**어떻게 쓰나?**
- 해외 매크로 뉴스, 글로벌 시장 동향 실시간 검색

### 10. 리스크관리 Specialist
**언제 쓰나?** 리스크 요인 모니터링
**어떻게 쓰나?**
- 지정학 리스크, 규제 변화 등 위험 요소 실시간 감시

### 11. 아카이브 Specialist
**언제 쓰나?** 아카이브 자료의 최신 관련 정보 확인
**어떻게 쓰나?**
- 기존 아카이브 문서와 연관된 최신 정보 검색

**실전 시나리오:**
> CEO가 "로스쿨 정원 확대 뉴스 있어?" 라고 하면:
> 1. `action=news, query="로스쿨 정원 확대 2026"` → 최신 뉴스 수집
> 2. AI가 뉴스를 종합 분석해서 핵심 트렌드와 시사점 정리
> 3. "2026년 2월 법무부 발표에 따르면..." 식의 최신 근거 제시

## web_search vs real_web_search 비교

| 항목 | web_search | real_web_search |
|------|-----------|-----------------|
| 방식 | AI 지식 기반 답변 | 실제 구글 검색 |
| 최신성 | AI 학습 시점까지 | 실시간 최신 |
| 속도 | 빠름 | 느림 (API 호출) |
| 비용 | 무료 | SerpAPI 유료 |
| API 키 | 불필요 | SERPAPI_KEY 필요 |
| 적합한 용도 | 기초 조사, 일반 지식 | 최신 뉴스, 실시간 데이터 |

## 주의사항
- `SERPAPI_KEY`가 설정되지 않으면 검색 불가
- SerpAPI 무료 플랜은 월 100회 제한 — 과도한 검색은 자제
- `httpx` 라이브러리가 설치되어 있어야 함
- 검색 결과는 구글 한국어(ko), 한국 지역(kr) 기준으로 설정됨
- 구글 검색 결과이므로 구글 정책에 따라 결과가 달라질 수 있음
