# sentiment_analyzer — SNS 감정 분석기 도구 가이드

## 이 도구는 뭔가요?
특정 키워드(브랜드명, 제품명, 이슈 등)에 대한 온라인 여론이 긍정적인지 부정적인지를 자동으로 분석하는 도구입니다. 네이버 뉴스와 블로그에서 관련 글을 수집하고, 한국어 감정 사전(긍정/부정 단어 목록)으로 감정을 판별합니다. AI가 여론 대응 전략까지 제안합니다.

## 어떤 API를 쓰나요?
- **네이버 검색 API** (https://developers.naver.com/) — 뉴스/블로그 검색
- 비용: **무료** (일일 25,000건 호출)
- 필요한 키: `NAVER_CLIENT_ID`, `NAVER_CLIENT_SECRET`

## 사용법

### action=analyze (키워드 감정 분석)
```
action=analyze, keyword="CORTHEX", sources="naver_news,naver_blog", count=50
```
- 특정 키워드에 대한 온라인 여론의 긍정/부정/중립 비율을 분석합니다.
- 자주 등장하는 긍정/부정 키워드, 대표 문장도 함께 제공합니다.
- AI가 여론 요약 + 긍정 강화 전략 + 부정 대응 전략을 제안합니다.

**파라미터:**
| 파라미터 | 설명 | 기본값 |
|---------|------|--------|
| keyword | 분석할 키워드 (필수) | - |
| sources | 데이터 소스 (쉼표 구분) | "naver_news,naver_blog" |
| count | 수집할 글 수 | 50 |

**예시:**
- `action=analyze, keyword="LEET Master"` → LEET Master 관련 여론 분석
- `action=analyze, keyword="AI 교육", count=100` → AI 교육 관련 100개 글 감정 분석

### action=trend (시간별 감정 추이)
```
action=trend, keyword="CORTHEX", days=30
```
- 특정 키워드의 감정이 시간에 따라 어떻게 변하고 있는지를 보여줍니다.
- 날짜별로 긍정/부정/중립 비율을 표로 정리합니다.
- 여론이 좋아지고 있는지, 나빠지고 있는지 추세를 파악할 수 있습니다.
- AI가 추이 분석과 향후 전망을 제공합니다.

**파라미터:**
| 파라미터 | 설명 | 기본값 |
|---------|------|--------|
| keyword | 분석할 키워드 (필수) | - |
| days | 분석 기간 (일) | 30 |

**예시:**
- `action=trend, keyword="LEET", days=7` → 최근 7일간 LEET 여론 추이
- `action=trend, keyword="AI 스타트업", days=60` → 2개월간 감정 변화

### action=report (종합 여론 보고서)
```
action=report, keyword="CORTHEX"
```
- 뉴스와 블로그를 모두 수집하여 종합 여론 보고서를 생성합니다.
- 뉴스 여론 / 블로그 여론 / 종합 분석을 구분하여 보여줍니다.
- 긍정/부정 키워드 Top 10을 제공합니다.
- AI가 CEO용 종합 보고서(핵심 요약, PR 액션 플랜, 중장기 전략)를 작성합니다.

**예시:**
- `action=report, keyword="CORTHEX"` → CORTHEX 종합 여론 보고서
- `action=report, keyword="법학적성시험"` → LEET 관련 종합 여론

## 이 도구를 쓰는 에이전트들

### 1. 마케팅/고객처장 (CMO)
**언제 쓰나?** 브랜드 평판을 모니터링하고 마케팅 전략을 수립할 때
**어떻게 쓰나?**
- report로 정기적인 브랜드 여론 보고서 생성
- trend로 캠페인 전후 여론 변화 추적
- 부정 여론이 급증하면 위기 대응 전략 수립

**실전 시나리오:**
> CEO가 "우리 브랜드 온라인 평판이 어때?" 라고 하면:
> 1. `action=report, keyword="CORTHEX"` 로 종합 여론 보고서 생성
> 2. "뉴스: 긍정 65%, 블로그: 긍정 55% — 전반적으로 긍정적이나 블로그에서 '가격' 관련 불만 5건 발견" 보고
> 3. AI가 제안한 PR 액션 플랜 3가지 전달

### 2. 설문/리서치 Specialist
**언제 쓰나?** 시장 조사, 경쟁사 분석, 사용자 리서치에서 여론 데이터가 필요할 때
**어떻게 쓰나?**
- analyze로 특정 제품/서비스에 대한 소비자 반응 파악
- 경쟁사 키워드를 분석하여 경쟁사 대비 우리의 여론 위치 파악

**실전 시나리오:**
> CEO가 "경쟁사 대비 우리 여론이 어때?" 라고 하면:
> 1. `action=analyze, keyword="우리 서비스명"` 로 우리 여론 분석
> 2. `action=analyze, keyword="경쟁사 서비스명"` 로 경쟁사 여론 분석
> 3. 두 결과의 긍정 비율을 비교하여 "우리 긍정률 60% vs 경쟁사 45% — 우세" 보고

### 3. 커뮤니티 Specialist
**언제 쓰나?** 커뮤니티/SNS에서 특정 이슈에 대한 반응을 파악할 때
**어떻게 쓰나?**
- analyze로 커뮤니티에서 화제가 된 이슈의 감정 분석
- trend로 이슈의 확산/진정 추이 모니터링
- 부정 여론 키워드를 파악하여 커뮤니티 대응 전략 수립

**실전 시나리오:**
> 커뮤니티에서 부정적 반응이 올라왔을 때:
> 1. `action=trend, keyword="이슈 키워드", days=7` 로 추이 확인
> 2. "3일 전부터 부정 비율 급증, '불만', '환불' 키워드 집중 — 위기 수준: 보통" 보고
> 3. AI 대응 전략에 따라 커뮤니티 공식 입장 게시

## 주의사항
- 감정 분석은 **한국어 감정 사전(80개 단어) 기반**입니다. 신조어, 은어, 반어법(비꼬기)은 정확하지 않을 수 있습니다.
- 네이버 검색 API는 **최대 1000건까지만 조회** 가능합니다 (start 값 제한).
- 분석 결과는 **수집 시점의 상위 검색 결과** 기반이므로, 전체 여론을 대표하지 않을 수 있습니다.
- `NAVER_CLIENT_ID`와 `NAVER_CLIENT_SECRET`이 설정되어 있어야 합니다. 네이버 개발자센터(https://developers.naver.com/)에서 무료 발급 가능합니다.
- 긍정 비율 60% 이상 → 긍정, 40% 미만 → 부정, 그 사이 → 중립으로 판정합니다.
