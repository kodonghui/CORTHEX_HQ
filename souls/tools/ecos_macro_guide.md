# ecos_macro — 한국은행 거시경제 지표 도구 가이드

## 이 도구는 뭔가요?
한국은행이 운영하는 ECOS(경제통계시스템)에서 금리, GDP, 물가, 환율 같은 경제 숫자를 직접 가져오는 도구입니다.
실제로 경제연구소나 은행의 이코노미스트(경제 분석가)가 매달 경제 지표를 추적하는 것과 같습니다.

## 어떤 API를 쓰나요?
- **한국은행 ECOS API** (ecos.bok.or.kr)
- 비용: **무료**
- 필요한 키: `ECOS_API_KEY` (한국은행 사이트에서 발급)

## 사용법

### action=indicator (경제 지표 조회)
```
action=indicator, indicator="지표명"
```
조회 가능한 주요 지표:
| 지표명 | 설명 | 단위 |
|---|---|---|
| 기준금리 | 한국은행이 정하는 기본 금리 | % |
| GDP | 국내총생산 (나라 전체 경제 크기) | 억원 |
| 소비자물가 | 물가 상승률 (물건값이 얼마나 올랐나) | % |
| 수출 | 해외로 팔은 물건 금액 | 백만달러 |
| 수입 | 해외에서 산 물건 금액 | 백만달러 |

**예시:**
- `action=indicator, indicator="기준금리"` → "현재 기준금리: 3.50%"
- `action=indicator, indicator="GDP"` → "2025년 GDP: 2,200조원"
- `action=indicator, indicator="소비자물가"` → "소비자물가 상승률: 2.3%"

### action=exchange_rate (환율 조회)
```
action=exchange_rate
```
- 원/달러, 원/엔, 원/유로 환율을 한번에 조회
- 별도 파라미터 불필요

**예시:**
- `action=exchange_rate` → "원/달러: 1,320원, 원/엔: 880원, 원/유로: 1,430원"

## 이 도구를 쓰는 에이전트

### 시황분석 Specialist (유일한 사용자)
**언제 쓰나?** 현재 경기 상태를 판단할 때
**어떻게 쓰나?**
- 기준금리 → "금리가 높으면 경기 억제, 낮으면 경기 부양 중"
- GDP → "경제가 커지고 있나 줄고 있나"
- 소비자물가 → "인플레이션이 심한가 안정되었나"
- 환율 → "원화 가치가 오르고 있나 떨어지고 있나"

**실전 시나리오:**
> CEO가 "지금 경기가 어떤 상태야?" 라고 물으면:
> 1. `action=indicator, indicator="기준금리"` → 3.50% (높은 편)
> 2. `action=indicator, indicator="GDP"` → 전년비 +2.1%
> 3. `action=indicator, indicator="소비자물가"` → +2.3% (안정 구간)
> 4. `action=exchange_rate` → 원/달러 1,320원
> 5. **종합 판단:** "금리 높고, GDP 성장 둔화, 물가 안정 → 경기 후반 국면. 하반기 금리 인하 가능성 있음"

**naver_news와 함께 쓰는 패턴:**
> 1. ecos_macro로 숫자 먼저 확인 (팩트)
> 2. naver_news로 시장 해석/전망 확인 (시장 분위기)
> 3. 숫자 + 시장반응을 종합해서 "현재 시장 국면" 판단

---

## 주의사항
- 경제 지표는 보통 1~3개월 전 데이터. 실시간이 아닙니다.
- GDP는 분기별, 물가는 월별로 발표됩니다.
- 환율은 비교적 최신 데이터지만, 실시간 시세와는 약간 차이가 있을 수 있습니다.
