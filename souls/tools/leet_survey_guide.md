# leet_survey — LEET 해설 서베이 도구 가이드

## 이 도구는 뭔가요?
LEET(법학적성시험) 해설에 대한 부정적 의견을 6개 커뮤니티에서 자동으로 수집하고 분석하는 도구입니다.
다음카페, 네이버카페, 네이버블로그, 오르비, 디시인사이드, 티스토리 — 이 6곳을 한 번에 뒤져서 "해설이 틀렸다", "해설이 부실하다" 같은 부정적 의견을 모아줍니다.
LEET Master 제품 개발을 위한 시장 조사 전문 도구입니다.

## 어떤 API를 쓰나요?
- **leet-opinion-scraper** (별도 프로젝트) — 6개 커뮤니티를 직접 크롤링(웹페이지 자동 수집)
- 비용: **무료** (단, 일부 플랫폼은 로그인 필요)
- 필요한 키:
  - `KAKAO_ID` / `KAKAO_PW` — 다음 카페 로그인용
  - `NAVER_ID` / `NAVER_PW` — 네이버 카페 로그인용
  - `ORBI_ID` / `ORBI_PW` — 오르비 로그인용 (선택)

## 사용법

### action=survey (커뮤니티 의견 수집)
```
action=survey, platforms="all", max_pages=3, keywords_only=true
```
- `platforms` (선택, 기본 "all"): 수집할 플랫폼. `all`(6개 전체) 또는 특정 플랫폼 지정 (쉼표 구분)
- `max_pages` (선택, 기본 3): 각 플랫폼에서 몇 페이지까지 검색할지
- `keywords_only` (선택, 기본 true): 키워드가 포함된 글만 수집할지
- `keywords` (선택): 에이전트가 직접 지정하는 검색 키워드 (쉼표 구분)
- `topic` (선택): 주제만 주면 AI가 자동으로 검색 키워드 15~25개를 생성

수집 후 AI가 자동 분석:
1. 핵심 불만 유형 분류 (해설 불일치, 오류, 부실 등)
2. 주요 불만 트렌드
3. 플랫폼별 의견 성향 차이
4. 사업 기회 분석
5. 경쟁사(메가, 피트, 진학사 등) 언급 분석
6. LEET Master 서비스 개선 제안

**예시:**
- `action=survey` → 6개 플랫폼 전체에서 기본 키워드로 수집
- `action=survey, platforms="dcinside,orbi", max_pages=2` → 디시+오르비만 빠르게 수집
- `action=survey, topic="2025 LEET 해설 품질"` → AI가 키워드 자동 생성 후 수집

### action=status (수집 현황 확인)
```
action=status
```
- 마지막으로 수집한 결과의 요약 정보를 보여줌
- 플랫폼별 수집 건수, 부정 의견 건수 등

### action=results (기존 결과 재분석)
```
action=results, file="결과파일경로"
```
- `file` (선택): 분석할 JSON 결과 파일 경로. 미지정 시 가장 최근 결과 사용
- 이미 수집된 데이터를 다시 분석할 때 사용 (다시 수집하지 않고 분석만)

## 이 도구를 쓰는 에이전트들

### 1. 커뮤니티 Specialist
**언제 쓰나?** LEET 해설에 대한 수험생 여론을 종합적으로 파악할 때
**어떻게 쓰나?**
- survey로 6개 플랫폼에서 한 번에 수집
- 수집 결과를 바탕으로 불만 유형을 분류하고 사업 기회 도출
- daum_cafe, naver_place_scraper와 함께 커뮤니티 종합 분석

**실전 시나리오:**
> CEO가 "수험생들이 현재 해설에 어떤 불만이 있는지 조사해줘" 라고 하면:
> 1. `action=survey, topic="LEET 해설 품질 불만"` → AI가 키워드 자동 생성 + 6개 플랫폼 수집
> 2. 수집된 부정 의견을 유형별로 분류 (해설 오류, 해설 불일치, 부실한 설명 등)
> 3. 경쟁사(메가, 피트 등) 관련 불만도 함께 분석
> 4. "이런 불만이 있으니 LEET Master에서는 이렇게 해결하면 됩니다" 제안

## 주의사항
- **leet-opinion-scraper가 별도로 설치되어 있어야 함** (프로젝트 루트의 `leet-opinion-scraper/` 디렉토리)
- 수집에 시간이 걸림: 6개 플랫폼 전체 수집 시 최대 **10분** (타임아웃 제한)
- 빠른 수집을 원하면 `platforms`를 줄이거나 `max_pages`를 낮추세요
- 다음카페/네이버카페는 로그인이 필요해서 `KAKAO_ID/PW`, `NAVER_ID/PW` 환경변수 설정 필요
- 결과 파일은 `leet-opinion-scraper/output/` 디렉토리에 JSON으로 저장됨
- 크롤링(자동 수집) 특성상 웹사이트 구조 변경 시 수집이 안 될 수 있음
