# scholar_scraper — 학술 논문 검색기 도구 가이드

## 이 도구는 뭔가요?
Google Scholar(구글 학술검색)에서 논문을 검색하고, 연도별 발표 추이를 분석하는 도구입니다.
"이 분야에 논문이 얼마나 나오고 있나?", "가장 많이 인용된 논문은?" 같은 학술 조사 질문에 답해줍니다.
사업에 활용할 수 있는 학술적 근거를 제공합니다.

## 어떤 API를 쓰나요?
- **Google Scholar 크롤링** (웹페이지 자동 수집) — scholarly 라이브러리 또는 직접 크롤링
- 비용: **무료**
- 필요한 키: 없음
- 선택 라이브러리: `scholarly` (있으면 더 정확, 없으면 `beautifulsoup4`로 대체)

## 사용법

### action=search (논문 검색)
```
action=search, query="키워드", count=20, year_from="2020", sort_by="relevance"
```
- `query` (필수): 검색할 학술 키워드
- `count` (선택, 기본 20, 최대 50): 검색 결과 개수
- `year_from` (선택): 특정 연도 이후 논문만 검색 (예: "2020")
- `sort_by` (선택): `relevance`(관련도순, 기본값) 또는 `date`(최신순)

반환 정보: 논문 제목, 저자, 연도, 인용 횟수, 초록(요약), 링크

**예시:**
- `action=search, query="법학적성시험 예측타당도"` → LEET 관련 학술 논문 검색
- `action=search, query="LEET prediction validity", year_from="2020"` → 2020년 이후 영문 논문

### action=cite (인용 정보 조회)
```
action=cite, title="논문 제목"
```
- `title` (필수): 인용 정보를 확인할 논문 제목
- scholarly 라이브러리가 있어야 정확한 인용 정보 제공 (없으면 검색으로 대체)

반환 정보: 제목, 저자, 연도, 저널, 인용 횟수, 초록, URL

**예시:**
- `action=cite, title="법학적성시험의 예측타당도 연구"` → 해당 논문 상세 인용 정보

### action=trend (연도별 논문 트렌드 분석)
```
action=trend, query="키워드", years=10
```
- `query` (필수): 트렌드를 분석할 키워드
- `years` (선택, 기본 10): 최근 몇 년간 트렌드를 볼 것인지

반환 정보: 연도별 논문 발표 건수 막대 차트 + AI 분석

AI 분석 내용:
1. 전체 추세 (증가/감소/유지)
2. 특이 시점 (급증/급감 구간)과 가능한 원인
3. 현재 연구 활성도 평가
4. 사업에 활용 가능한 학술 근거 및 시사점

**예시:**
- `action=trend, query="LEET 법학적성시험", years=10` → 최근 10년 논문 추이
- `action=trend, query="AI education assessment"` → AI 교육 평가 분야 트렌드

## 이 도구를 쓰는 에이전트들

### 1. 사업기획처장 (CSO)
**언제 쓰나?** 사업 기획 시 학술적 근거 확보
**어떻게 쓰나?**
- 시장 규모 추정의 학술적 근거 확보
- 제품 개발 방향의 연구 기반 검증
- trend로 연구 활성도 파악 → 성장하는 분야인지 판단

### 2. 시장조사 Specialist
**언제 쓰나?** 시장 조사 보고서에 학술 자료 인용
**어떻게 쓰나?**
- 시장 분석에 학술 연구 결과를 근거로 제시
- platform_market_scraper(시장 데이터)와 학술 데이터를 함께 제시하면 설득력 UP

### 3. 설문/리서치 Specialist
**언제 쓰나?** 설문 설계 시 선행연구 조사
**어떻게 쓰나?**
- 설문 문항 설계의 학술적 근거 확보
- 기존 연구의 방법론 참고

**실전 시나리오:**
> CEO가 "LEET 해설 품질이 합격에 영향을 미친다는 근거가 있나?" 라고 하면:
> 1. `action=search, query="LEET 해설 학습효과", year_from="2018"` → 관련 논문 검색
> 2. `action=search, query="법학적성시험 예측타당도"` → LEET 자체 연구 검색
> 3. `action=trend, query="법학적성시험", years=10` → 연구 추이 확인
> 4. "2020년 OOO 교수 연구에 따르면 해설 품질이 학습 효과에 유의미한 영향..." 식으로 근거 제시

## 주의사항
- Google Scholar는 대량 요청 시 **요청 제한(429 에러)**을 걸 수 있음 — 잠시 후 재시도 필요
- 요청 간 1~2초 딜레이가 자동으로 적용되므로 검색에 시간이 걸림
- `scholarly` 라이브러리가 있으면 더 정확한 결과를 얻을 수 있음 (없으면 HTML 크롤링으로 대체)
- trend 분석은 연도별로 검색하므로 10년 분석 시 10~20초 소요
- 영어/한국어 키워드 모두 검색 가능하지만, 한국어 논문은 Google Scholar에 등록된 것만 검색됨
- 인용 수가 높은 논문이 반드시 좋은 논문은 아님 — AI 분석과 함께 종합적으로 판단하세요
