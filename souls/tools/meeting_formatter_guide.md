# meeting_formatter — 회의록 자동 정리기 도구 가이드

## 이 도구는 뭔가요?
회의 내용을 넣으면 자동으로 깔끔한 회의록으로 정리해주는 도구입니다.
회의에서 나온 결정사항, 해야 할 일(Action Items), 담당자, 기한을 자동으로 뽑아내고,
회의 유형에 맞는 구조화된 회의록을 생성합니다.

## 어떤 API를 쓰나요?
- **내부 정규식(패턴 매칭) + LLM 분석**
- "해야 합니다", "까지 완료", "담당: OO" 같은 패턴을 자동 감지합니다
- AI가 전체 맥락을 이해하여 구조화된 회의록을 생성합니다
- 비용: **무료** (패턴 매칭 무료, AI 정리에 약간의 API 비용)
- 필요한 키: 없음

## 사용법

### action=format (회의 내용을 구조화된 회의록으로 정리)
```
action=format, text="회의 내용 전체 텍스트...", meeting_type="일반"
```
- 회의 내용 텍스트를 받아서 구조화된 회의록으로 변환합니다
- `text`: 회의 내용 (필수) — 녹음 풀어쓴 것, 메모, 채팅 로그 등 아무 형식이나 가능
- `meeting_type`: 회의 유형 (기본값: "일반")
  - "일반": 일반 비즈니스 회의
  - "투자검토": 투자 종목 검토 회의
  - "기획회의": 프로젝트 기획 회의
  - "기술회의": 기술 관련 회의
- AI가 회의 내용에서 참석자, 안건, 논의 사항, 결정 사항, 할 일, 다음 회의를 자동 추출합니다

**예시:**
- `action=format, text="오늘 삼성전자 투자 검토했는데...", meeting_type="투자검토"` → 투자 검토 회의록 형식으로 정리
- `action=format, text="프론트엔드 리팩토링 논의...", meeting_type="기술회의"` → 기술 회의록 형식으로 정리

### action=action_items (할 일 목록만 추출)
```
action=action_items, text="회의 내용 전체 텍스트..."
```
- 회의 내용에서 해야 할 일(Action Items)만 뽑아냅니다
- `text`: 회의 내용 (필수)
- 2단계로 추출합니다:
  1. 정규식(패턴 매칭)으로 "해야 합니다", "까지 완료", "담당:", "TODO" 등의 문장 감지
  2. AI가 전체 맥락에서 추가 할 일 추출
- 각 할 일에 담당자와 기한이 있으면 함께 표시합니다

**예시:**
- `action=action_items, text="김대리가 다음주까지 시장조사 보고서 제출해야 하고, 박과장은 예산안 준비..."` → 할 일 목록 추출

### action=template (회의록 빈 양식 제공)
```
action=template, meeting_type="기획회의"
```
- 회의 유형별 빈 양식(템플릿)을 제공합니다
- `meeting_type`: 양식 유형 (기본값: "일반")
- 사용 가능한 양식: "일반", "투자검토", "기획회의", "기술회의"
- 양식을 복사하여 직접 내용을 채울 수 있습니다

**사용 가능한 회의록 양식:**
| 양식명 | 용도 | 포함 섹션 |
|-------|------|---------|
| 일반 | 일반 비즈니스 회의 | 기본정보, 안건, 논의사항, 결정사항, Action Items, 다음회의 |
| 투자검토 | 투자 종목 검토 | 검토종목 표, 시장환경, Bull/Bear Case, 결정사항, 후속조치 |
| 기획회의 | 프로젝트 기획 | 현황보고, 목표/범위/일정/리소스, 결정사항, 마일스톤 |
| 기술회의 | 기술 관련 | 기술안건, 아키텍처 논의, 기술결정 표, 코드리뷰, 기술부채 |

## 이 도구를 쓰는 에이전트들

### 1. 출판/기록처장 (CPO)
**언제 쓰나?** 회의 내용을 공식 기록으로 남길 때, 임원 회의록 정리 시
**어떻게 쓰나?**
- `action=format`으로 회의 내용을 구조화된 회의록으로 변환
- `action=template`으로 표준 양식 제공

**실전 시나리오:**
> CEO가 회의 내용을 보내며 "회의록 정리해줘" 라고 하면:
> 1. `action=format, text="회의 내용...", meeting_type="일반"` 실행
> 2. 자동 생성된 회의록을 검토/수정
> 3. 결정 사항과 할 일을 CEO에게 요약 보고

### 2. 콘텐츠편집 Specialist
**언제 쓰나?** 회의 기록을 정리된 문서로 변환할 때
**어떻게 쓰나?**
- `action=action_items`로 할 일 목록 추출 → 팀에 공유
- `action=format`으로 회의록 편집 후 최종 산출물 생성

**실전 시나리오:**
> 기술 회의 후 회의록을 정리할 때:
> 1. `action=format, text="...", meeting_type="기술회의"` 로 초안 생성
> 2. 기술 용어에 쉬운 설명을 추가하여 CEO용 버전 작성
> 3. `action=action_items`로 할 일만 별도 추출하여 진행 추적

## 주의사항
- 회의 내용(`text`)에 없는 정보는 AI가 만들어내지 않습니다 (사실만 추출)
- 한국어 패턴을 기반으로 감지하므로, 영어 회의 내용은 정규식 감지가 약할 수 있습니다 (AI 추출은 가능)
- 감지하는 패턴: "해야 합니다", "까지 완료/제출/보고", "담당:", "기한:", "TODO", "확인 필요", "준비 필요" 등
- 회의록 양식의 날짜는 자동으로 현재 날짜로 채워집니다
- 매우 긴 회의 내용(수만 자)은 AI 분석 시 일부가 잘릴 수 있습니다
