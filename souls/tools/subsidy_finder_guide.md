# subsidy_finder — 정부 지원금 찾기 도구 가이드

## 이 도구는 뭔가요?
기업마당(bizinfo.go.kr)에서 정부 지원사업을 자동으로 검색하고, 우리 회사 조건에 맞는 지원사업을 찾아주는 도구입니다.
"우리 회사가 받을 수 있는 정부 지원금이 뭐가 있지?", "교육 스타트업이 신청할 수 있는 지원사업은?" 같은 질문에 답해줍니다.
AI가 검색 결과를 분석해서 가장 유망한 지원사업을 추천하고 신청 전략까지 제안합니다.

## 어떤 API를 쓰나요?
- **기업마당 RSS 피드 + 웹 검색** (https://www.bizinfo.go.kr)
- 비용: **무료**
- 필요한 키: 없음
- 의존 라이브러리: `httpx`, `beautifulsoup4`

## 사용법

### action=search (지원사업 검색)
```
action=search, keyword="검색어", category="전체", region="전국", count=20
```
- `keyword` (필수): 검색할 키워드 (예: "AI 교육 스타트업", "법학 교육")
- `category` (선택, 기본 "전체"): 지원사업 분류
- `region` (선택, 기본 "전국"): 지역 필터
- `count` (선택, 기본 20, 최대 50): 검색 결과 개수

기업마당 RSS 피드와 웹 검색 두 가지 경로로 검색한 뒤, AI가 분석합니다:
1. 가장 유망한 지원사업 TOP 3 추천
2. 각 지원사업의 지원 규모 및 자격 조건
3. 신청 전략 및 준비 사항
4. 주의사항 (마감일, 제한 조건 등)

**예시:**
- `action=search, keyword="AI 교육"` → AI 교육 관련 정부 지원사업 검색
- `action=search, keyword="창업 지원", region="서울"` → 서울 지역 창업 지원사업

### action=detail (지원사업 상세 정보)
```
action=detail, url="지원사업 상세페이지 URL"
```
- `url` (필수): 지원사업 상세 페이지 주소
- 페이지 내용을 읽어와서 AI가 구조화된 형태로 정리

정리 항목:
1. 사업명
2. 지원 기관
3. 지원 규모 (금액)
4. 지원 대상 (자격 조건)
5. 신청 기간
6. 지원 내용 상세
7. 신청 방법
8. 참고 사항

**예시:**
- `action=detail, url="https://www.bizinfo.go.kr/..."` → 해당 지원사업 상세 분석

### action=match (우리 회사 맞춤 매칭)
```
action=match, company_type="창업3년이내", industry="교육"
```
- `company_type` (선택): 기업 유형. 옵션: `예비창업자`, `창업3년이내`, `창업7년이내`, `중소기업`
- `industry` (선택): 업종. 예: `교육`, `IT`, `서비스`, `제조`, `바이오`
- 둘 중 최소 하나는 입력해야 함

AI가 회사 조건에 맞는 지원사업을 매칭하고 분석합니다:
1. 적합한 지원사업을 적합도(높음/중간/낮음)로 정렬
2. 우선 신청해야 할 지원사업 TOP 3
3. 신청 준비 로드맵 (시간순)

**예시:**
- `action=match, company_type="창업3년이내", industry="교육"` → 교육 스타트업 맞춤 지원사업
- `action=match, industry="IT"` → IT 업종 맞춤 지원사업

## 이 도구를 쓰는 에이전트들

### 1. 사업기획처장 (CSO)
**언제 쓰나?** 사업 자금 확보 전략 수립 시
**어떻게 쓰나?**
- match로 우리 회사 맞춤 지원사업 파악
- search로 최신 지원사업 검색
- financial_calculator(재무 계산)와 함께 자금 계획 수립

### 2. 사업계획서 Specialist
**언제 쓰나?** 사업계획서 작성 시 활용 가능한 지원사업 조사
**어떻게 쓰나?**
- 사업계획서에 "정부 지원사업 활용 계획" 섹션 작성 시
- detail로 지원사업 요건을 상세 파악하고 사업계획서에 반영

**실전 시나리오:**
> CEO가 "우리가 받을 수 있는 정부 지원금 알아봐" 라고 하면:
> 1. `action=match, company_type="창업3년이내", industry="교육"` → 맞춤 지원사업 검색
> 2. TOP 3 지원사업의 URL로 `action=detail` → 상세 요건 확인
> 3. "1순위: OO 지원사업 (최대 1억원, 마감 3/31), 준비물: 사업계획서, IR 자료" 식으로 보고
> 4. 신청 준비 일정표도 함께 제안

## 주의사항
- 기업마당 웹사이트 구조 변경 시 검색이 안 될 수 있음
- 기업마당 RSS 피드에 모든 지원사업이 포함되지는 않음 — 웹 검색도 병행
- 지원사업 상세 정보(자격 조건, 마감일 등)는 반드시 `action=detail`로 원본 페이지를 확인하세요
- AI 분석은 참고용이며, 실제 신청 자격은 원본 공고를 직접 확인해야 합니다
- `beautifulsoup4` 라이브러리가 설치되어 있어야 함
- 검색 결과가 없으면 다른 키워드로 재시도하거나 기업마당(https://www.bizinfo.go.kr) 직접 방문 권장
