# naver_news — 네이버 뉴스 검색 도구 가이드

## 이 도구는 뭔가요?
네이버에서 뉴스 기사를 자동으로 검색해서 결과를 정리해주는 도구입니다.
실제로 증권사 애널리스트(주식 분석가)가 매일 아침 뉴스를 모아보는 것과 같은 일을 합니다.

## 어떤 API를 쓰나요?
- **네이버 검색 API** (openapi.naver.com)
- 비용: **무료** (하루 25,000건까지)
- 필요한 키: `NAVER_CLIENT_ID`, `NAVER_CLIENT_SECRET`

## 사용법

### action=search (일반 뉴스 검색)
```
action=search, query="검색어"
```
- 네이버 뉴스에서 키워드로 검색
- 최신순으로 최대 10개 기사 반환
- 각 기사의 제목, 요약, 날짜, 링크 제공

**예시:**
- `action=search, query="반도체 시장 전망"` → 반도체 관련 최신 뉴스
- `action=search, query="삼성전자 실적"` → 삼성전자 실적 관련 기사
- `action=search, query="금리 인상"` → 금리 관련 뉴스

### action=finance (금융 뉴스 전문 검색)
```
action=finance, query="키워드"
```
- 금융/경제 전문 뉴스만 필터링해서 검색
- 일반 뉴스보다 투자/경제에 더 집중된 결과

**예시:**
- `action=finance, query="코스피"` → 주식시장 관련 금융 뉴스
- `action=finance, query="환율 전망"` → 환율 관련 전문 기사

## 이 도구를 쓰는 에이전트들

### 1. 시황분석 Specialist
**언제 쓰나?** 시장 전체 분위기를 파악할 때
**어떻게 쓰나?**
- "금리", "환율", "코스피", "미국 연준" 등으로 검색
- ecos_macro(경제 지표)와 함께 써서 "숫자 + 뉴스"로 종합 판단
- **순서:** ecos_macro로 지표 먼저 확인 → naver_news로 시장 반응 확인

**실전 시나리오:**
> CEO가 "요즘 시장 어때?" 라고 물으면:
> 1. `ecos_macro action=indicator, indicator="기준금리"` → 현재 금리 확인
> 2. `naver_news action=finance, query="금리 동결"` → 관련 뉴스 확인
> 3. "기준금리 3.5% 유지 중, 시장은 하반기 인하 기대감으로 상승 분위기"

### 2. 종목분석 Specialist
**언제 쓰나?** 특정 기업의 최신 이슈를 파악할 때
**어떻게 쓰나?**
- 기업명으로 검색해서 최근 뉴스/이슈 파악
- dart_api(재무제표)와 함께 써서 "숫자 + 뉴스"로 종합 판단
- **순서:** dart_api로 재무 먼저 확인 → naver_news로 최근 이슈 확인

**실전 시나리오:**
> CEO가 "삼성전자 분석해줘" 라고 하면:
> 1. `kr_stock action=price, name="삼성전자"` → 현재 주가 확인
> 2. `dart_api action=financial, company="삼성전자"` → 재무제표 확인
> 3. `naver_news action=search, query="삼성전자 실적"` → 최근 뉴스 확인
> 4. 종합: "현재가 7만원, 영업이익 전년 대비 200% 증가, AI 반도체 수요 증가 뉴스"

### 3. CIO 처장 (투자분석처장)
**언제 쓰나?** 투자 판단의 최종 검토 단계에서
**어떻게 쓰나?**
- Specialist들의 분석 결과를 받은 후, 직접 뉴스를 확인하여 교차 검증
- 주로 큰 이슈(금융 위기, 정책 변화 등) 확인용

---

## 주의사항
- 네이버 뉴스는 한국 뉴스 위주. 글로벌 뉴스는 web_search 도구를 병행하세요.
- 검색 결과는 네이버가 정렬한 것이므로 편향이 있을 수 있습니다.
- 속보성 뉴스는 정확도가 떨어질 수 있으니 여러 기사를 비교하세요.
