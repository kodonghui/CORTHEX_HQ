# hashtag_recommender — 해시태그 추천기 도구 가이드

## 이 도구는 뭔가요?
SNS 게시물에 달 해시태그를 자동으로 추천해주는 도구입니다.
게시물 주제와 플랫폼(인스타그램/유튜브/틱톡)에 맞춰 대형/중형/소형 해시태그를 최적 비율로 조합하여 노출 효과를 극대화합니다.

## 어떤 API를 쓰나요?
- **내장 해시태그 데이터베이스** (교육/금융/기술/라이프스타일 4개 분야)
- **인스타그램 웹 스크래핑** (해시태그별 게시물 수 조회, analyze 액션)
- **AI 분석** (추가 해시태그 추천, 전략 조언)
- 비용: **무료**
- 필요한 키: 없음

## 사용법

### action=recommend (해시태그 추천)
```
action="recommend", topic="LEET 공부법", platform="instagram", count=30
```
- 주제에 맞는 해시태그를 대형/중형/소형 비율로 자동 조합하여 추천합니다
- 내장 데이터베이스에서 분야를 자동 매칭하고, 주제 키워드로 커스텀 해시태그도 생성합니다
- AI가 게시 최적 시간대, 추가 해시태그, 주의사항을 분석합니다

**플랫폼별 해시태그 규칙:**

| 플랫폼 | 최대 개수 | 최적 개수 | 대형:중형:소형 비율 |
|--------|----------|----------|-------------------|
| Instagram | 30개 | 20~30개 | 10%:60%:30% |
| YouTube | 15개 | 5~15개 | 20%:50%:30% |
| TikTok | 10개 | 3~8개 | 30%:50%:20% |

**예시:**
- `action="recommend", topic="LEET 공부법", platform="instagram"` → 30개 해시태그 세트 (대형 3개, 중형 18개, 소형 9개) + AI 전략

### action=analyze (해시태그 인기도 분석)
```
action="analyze", hashtags="#LEET,#로스쿨,#법학"
```
- 지정한 해시태그의 인스타그램 게시물 수를 조회하여 규모(대형/중형/소형)와 경쟁도를 분석합니다
- 100만 이상 = 대형(경쟁 높음), 1만~100만 = 중형(보통), 1만 미만 = 소형(경쟁 낮음)
- AI가 각 해시태그의 효용성, 함께 사용할 전략, 대체 해시태그를 추천합니다

**예시:**
- `action="analyze", hashtags="#리트공부,#LEET준비,#로스쿨입시"` → 각 해시태그별 게시물 수, 규모, 경쟁도 표

### action=trending (인기 해시태그 목록)
```
action="trending", category="교육"
```
- 카테고리별 인기 해시태그 목록을 대형/중형/소형으로 분류하여 보여줍니다
- category를 지정하지 않으면 전체 카테고리(교육/금융/기술/라이프스타일) 목록을 표시합니다
- AI가 현재 성장 중인 트렌드, 효과적인 조합, 새로 떠오르는 해시태그를 분석합니다

**사용 가능 카테고리:** 교육, 금융, 기술, 라이프스타일

**예시:**
- `action="trending", category="금융"` → 금융 분야 대형/중형/소형 해시태그 전체 목록

## 이 도구를 쓰는 에이전트들

### 1. CMO 마케팅처장 (cmo_manager)
**언제 쓰나?** SNS 콘텐츠 발행 전 최적 해시태그 전략을 세울 때
**어떻게 쓰나?**
- recommend로 게시물별 해시태그 세트 생성
- analyze로 기존 사용 중인 해시태그 효과 검증
- trending으로 분야별 인기 해시태그 트렌드 파악

**실전 시나리오:**
> CEO가 "인스타그램에 LEET 관련 게시물 올려야 하는데 해시태그 뭐 써야 해?" 라고 하면:
> 1. `action="recommend", topic="LEET 로스쿨 준비", platform="instagram"` 실행
> 2. 추천된 30개 해시태그 세트를 콘텐츠 Specialist에게 전달
> 3. 게시물에 해시태그 적용 후 sns_manager로 발행 요청

## 주의사항
- analyze 액션은 인스타그램 웹 스크래핑을 사용하므로, 차단 방지를 위해 요청 간 2.5초 딜레이가 있습니다 (해시태그가 많으면 시간이 오래 걸림)
- 인스타그램이 접근을 차단하면 게시물 수 조회가 실패할 수 있음 (조회 불가로 표시)
- 내장 데이터베이스는 한국 시장 기준이므로 해외 타겟 마케팅에는 적합하지 않을 수 있음
- 해시태그 트렌드는 빠르게 변하므로 주기적으로 analyze로 재검증하는 것을 권장
