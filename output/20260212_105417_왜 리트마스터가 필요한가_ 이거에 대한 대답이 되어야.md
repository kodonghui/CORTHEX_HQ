#### 상세 사례: 문제점 1 (정답 도출 과정의 비약)

**기존 해설 (메가로스쿨):**
```
【문제】
A, B, C 세 사람이 있고, 다음 조건을 만족한다:
(1) A가 참석하면 B도 참석한다.
(2) B와 C는 동시에 참석할 수 없다.
(3) C가 참석하지 않으면 A는 참석한다.

【메가로스쿨 해설】
조건 (1): A → B
조건 (2): ¬(B ∧ C)
조건 (3): ¬C → A

따라서 정답은 ③번이다.

❌ 문제점:
- 기호 표기만 있고 풀이 과정 생략
- 경우의 수 분석 없음
- "왜 ③번인지" 논리적 연결 부재
```

**LEET MASTER 해설 (LEVEL 2):**
```
【LEET MASTER 해설】

🔍 조건 분석
(1) A → B      (A 참석 → B 참석)
(2) ¬(B ∧ C)   (B, C 동시 불가)
(3) ¬C → A     (C 미참석 → A 참석)

📊 경우의 수 분석
┌──────┬──────┬──────┬──────────┐
│ A    │ B    │ C    │ 가능?    │
├──────┼──────┼──────┼──────────┤
│ O    │ O    │ X    │ ✓ 가능   │
│ O    │ O    │ O    │ ✗ 조건(2) 위반 │
│ X    │ X    │ O    │ ✓ 가능   │
│ X    │ X    │ X    │ ✗ 조건(3) 위반 │
└──────┴──────┴──────┴──────────┘

✅ 결론
가능한 경우: (A O, B O, C X), (A X, B X, C O)
따라서 정답은 ③번입니다.

💡 학습 포인트
- 조건부 명제는 표로 정리하면 실수 방지
- 모든 경우의 수를 체계적으로 검토
```
