## 1. "지금 분석" + 매매 결정 일지 (2026-02-19 확정, C안)

### 배경
- 대시보드에 "지금 분석" 버튼이 있음
- CEO 의도: "지금 당장 시장 분석 → 매수/매도/관망 결정 → 근거 보고서"까지 한 세트
- 단순 시그널 생성이 아님

### 확정된 설계 (C안)
1. **버튼**: 대시보드(작전현황)에 "지금 분석" 버튼 유지
2. **동작**: 버튼 누르면 → CIO가 전문가 4명에게 분석 지시 → 매수/매도/관망 결정 → 근거 보고서 작성
3. **대시보드에 요약**: 최신 1건 분석 결과 요약 카드 표시
4. **시그널 탭에 이력**: "매매 결정 일지" — 과거 모든 분석 결과 이력 관리
5. **전략실 협업로그**: 분석 과정(CIO↔전문가 소통)이 실시간으로 보이는 로그

### 흐름
```
대시보드 "지금 분석" 클릭
  → CIO가 전문가 4명에게 분석 지시 (시황/종목/기술/리스크)
  → 전략실 협업로그에 분석 과정 표시
  → CIO가 전문가 결과 종합 → 매수/매도/관망 결정
  → 시그널 탭 "매매 결정 일지"에 보고서 등록
  → 대시보드에 최신 1건 요약 카드 업데이트
```

## 2. 전략실 협업로그 (2026-02-19 확정)

- 전략실(Trading Ops) 전용 로그
- CIO + 전문가 4명(시황분석가/종목분석가/기술분석가/리스크관리자) 분석 과정 표시
- 내부통신과 유사하지만 투자 분석에 특화
- "지금 분석" 기능과 자연스럽게 연결

## 3. P2P 에이전트 소통 + 내부통신 개편 (2026-02-19 확정)

### 핵심 결정
- **cross_agent_protocol 도구(또는 유사 P2P 도구)를 전 직원(29명)이 사용 가능하게 할 것**
- 현재는 파일 저장만 → 실시간 DB 저장 + WebSocket 알림으로 업그레이드
- 사령관실 내부통신 탭에서 P2P 메시지도 표시

### 배경 (2026-02-17 협업 구조 개편 당시 결정)
- A안: 소통 채널 추가 (처장↔처장, 전문가↔전문가 직접)
- C+안: 자율 작업 + 보고서 1개
- 스마트 라우팅: Level 1~4 복잡도별 자동 조정
- → 2.00.000에서 구현 완료
- **남은 것**: P2P 도구를 실제로 전 직원에게 열고, 내부통신에서 보여주는 것
