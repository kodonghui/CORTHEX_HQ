# 에이전트 구조조정 — 비서실/CTO/법무처

- **날짜**: 2026-02-21
- **브랜치**: claude/agent-restructure
- **버전**: 3.02.018

---

## 변경 내용

### 1. 비서실 — 정보보좌관 뉴스 특화
- 모델: Gemini 2.5 Flash low → **Gemini 3.1 Pro high**
- 프롬프트에 "뉴스 브리핑" 핵심 임무 추가
- 이미 보유한 web_search/naver_news 도구로 뉴스 검색 가능
- 매일 09:00 뉴스 cron은 이미 구현되어 있었음 (SerpAPI → 텔레그램)

### 2. CTO부서 5명 — 동면 모드
- CTO: Opus high → **Flash low**
- 프론트/백엔드/인프라/AI모델: Pro high → **Flash low**
- 사유: 리트마스터 개발 중단 중이라 현재 쓸 일 없음
- 리트마스터 재개 시 원래 모델로 복귀 예정

### 3. 법무처 3명 — 한국법 특화 + 변호사급 전면 교체
- **EU법/GDPR/AI Act 등 외국법 관련 내용 전부 삭제**
- CEO가 법학 전공자임을 명시 → 전문 용어 번역 없이 조문·판례 직접 제시
- CLO 추가 법령:
  - 상법·회사법 (법인설립, 정관, 이사회, 상업등기)
  - 세법 실무 (사업자등록, 부가세, 원천징수, 세금계산서, 4대보험, 예비창업자 세제혜택)
- 저작권 Specialist: 한국 저작권법 조문 기반 실무 분석
- 특허/약관 Specialist: 특허출원 비용·기간, 약관규제법 실무 절차

### 4. 일정보좌관 모델 상향 (이전 브랜치)
- Flash low → **Gemini 3.1 Pro high** (신문 크롤링 대비)

## 비용 영향
- CTO부서 5명 동면: Opus 1명 + Pro 4명 → Flash 5명 = **크레딧 대폭 절감**
- 정보보좌관 상향: Flash → Pro = 약간 증가 (Gemini 무료 크레딧)
- 법무처: 모델 변경 없음 (Sonnet medium 유지)
