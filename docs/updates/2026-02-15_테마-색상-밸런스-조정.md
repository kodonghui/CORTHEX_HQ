# 다크/라이트 모드 색상 밸런스 조정 — 가시성 개선

## 📝 변경 사항 요약

### 문제점
- **다크 모드**: 배경색이 너무 어두워서 화면이 거의 검은색으로 보임
- **라이트 모드**: 배경과 패널의 색상 대비가 너무 낮아서 요소들이 잘 구분되지 않음
- CEO 피드백: "다크 모드는 너무 다크하고, 라이트 모드는 너무 밝아서 아무것도 안 보여"

### 해결 방법
웹 대시보드의 CSS 변수 색상 값을 조정했습니다.

#### 다크 모드 (어두운 테마)
화면을 약간 밝게 조정해서 가독성을 높였습니다.
- 배경색: `rgb(8, 14, 28)` → `rgb(26, 35, 50)` (약 15% 밝아짐)
- 서페이스: `rgb(18, 32, 58)` → `rgb(39, 52, 71)` (약 10% 밝아짐)
- 패널: `rgb(30, 50, 82)` → `rgb(52, 66, 86)` (약 10% 밝아짐)

#### 라이트 모드 (밝은 테마)
배경과 패널의 대비를 높여서 요소 구분이 명확하게 보이도록 했습니다.
- 배경색: `rgb(238, 242, 248)` → `rgb(241, 245, 249)` (약간 어둡게)
- 패널: `rgb(220, 228, 240)` → `rgb(226, 232, 240)` (약간 어둡게, 대비 향상)

### 수정한 파일
- `web/templates/index.html` (웹 대시보드 메인 화면) — CSS 변수 색상 값 수정
