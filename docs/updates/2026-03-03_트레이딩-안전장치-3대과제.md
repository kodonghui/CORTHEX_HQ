## 과제 1: 트리거 무한누적 방지

### 1A. 새 분석 시 기존 buy_limit 전부 초기화
- **위치**: `_run_trading_now_inner()` buy_limit 등록 루프 직전
- **동작**: 새 분석 결과가 나오면 기존 활성 buy_limit 전부 `active=False` 처리
- stop_loss/take_profit은 건드리지 않음 (기존 포지션 보호용)

### 1B. 10분 미체결 buy_limit 자동 만료

- **동작**: `created_at` 기준 600초(10분) 초과한 buy_limit → 자동 비활성화
- 로그: "⏰ buy_limit N건 10분 만료 처리"

## 과제 3: 전체자산 기반 비중 계산 + 100만원 폴백 제거

### 변경 전
```
account_balance = _bal.get("cash", 0)  # 현금만
if account_balance <= 0:
    account_balance = 1_000_000  # 위험한 폴백
```

### 변경 후
```
account_balance = _bal.get("total_eval", 0) or _bal.get("cash", 0)  # 총자산
available_cash = _bal.get("cash", 0)  # 가용 현금
_order_amt = min(목표금액, available_cash)  # 현금 캡 적용
# 100만원 폴백 제거 → 잔고 조회 실패 시 매매 건너뜀
```

### buy_limit 트리거 등록에도 동일 적용
- 잔고 0이면 트리거 등록 자체를 건너뜀 (`continue`)
