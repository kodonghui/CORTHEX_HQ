# 작업내역 탭 + 명령 입력 UI 개선

## 버전
0.05.052

## 작업 날짜
2026-02-15

## 작업 브랜치
claude/improve-history-command-ui

## 변경 사항 요약

### 1. 통계 카드 디자인 개선 (작업내역 탭 상단)
- **변경 전**: 회색 네모 4개에 숫자만 있어서 밋밋했음
- **변경 후**: 각 카드에 아이콘 + 왼쪽 색상 바 + 큰 숫자로 한눈에 구분 가능
  - 전체 작업: 파란색 클립보드 아이콘
  - 완료: 초록색 체크 아이콘
  - 실패: 빨간색 경고 아이콘
  - 총 비용: 시안색 달러 아이콘

### 2. 명령 입력 영역 개선
- **그라데이션 글로우**: 입력창에 글을 쓰려고 클릭하면 파랑→보라 색이 번갈아 빛나는 효과
- **힌트 순환**: 입력창이 비어있을 때 3초마다 예시 명령어가 자동으로 바뀜
  - "명령을 입력하세요..." → "@CTO 기술 현황 리포트 작성해줘" → "@CMO 마케팅 전략 분석해줘" 등
  - 사용자가 어떤 명령을 쓸 수 있는지 자연스럽게 알려주는 효과

### 3. 빈 상태 화면 개선 (작업내역이 비어있을 때)
- **변경 전**: "📋 작업 내역이 없습니다" 텍스트만 있어서 허전했음
- **변경 후**:
  - 둥둥 떠다니는 모니터 아이콘 (CSS 애니메이션)
  - 깜빡이는 장식 점 (활동 중인 느낌)
  - "에이전트들이 대기 중이에요" 메시지
  - 빠른 명령 버튼 3개 (업무현황/기술리포트/마케팅분석) — 클릭하면 명령 탭으로 이동 + 자동 입력

### 4. 샘플 데이터 추가
- API에서 데이터를 못 가져올 때 (서버가 아직 지원 안 하니까) → 자동으로 샘플 데이터 6개 표시
- 샘플 데이터 종류:
  - CTO 기술 리포트 (완료, 북마크됨)
  - 전 부서 업무 현황 보고 (완료)
  - CMO 마케팅 분석 (완료)
  - CFO 예산 정리 (실패 — API 키 만료)
  - CLO 법률 검토 (완료, 북마크됨)
  - CSO 보안 스캔 (진행중)
- 상단에 노란색 "미리보기용 샘플 데이터입니다" 배너 표시 → 진짜 데이터가 아님을 명확히 알림
