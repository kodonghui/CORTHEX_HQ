# CIO 투자분석처 신규 도구 5개 추가

## 작업 날짜
2026-02-14

## 작업 브랜치
claude/cio-investment-tools-e3Zo9

## 변경 사항 요약

투자분석처(CIO) 소속 에이전트들이 사용할 수 있는 전문 투자 분석 도구 5개를 새로 만들었습니다.

### 새로 만든 파일

| 파일 | 설명 |
|------|------|
| `src/tools/dart_monitor.py` (공시 알리미) | 관심 기업을 등록해두면 새 공시(기업이 금융감독원에 제출하는 보고서)가 올라올 때 자동으로 알려주고, 그 공시가 주가에 미치는 영향을 분석해줍니다 |
| `src/tools/stock_screener.py` (종목 스크리너) | "PER 10 이하, 시가총액 1000억 이상" 같은 조건을 넣으면 한국 전체 상장사 중에서 조건에 맞는 종목만 골라줍니다. 가치주/성장주/배당주/모멘텀 프리셋 전략도 있습니다 |
| `src/tools/backtest_engine.py` (포트폴리오 백테스터) | "삼성전자에 골든크로스 전략을 적용했다면 1년간 수익률이 얼마였을까?"를 과거 데이터로 시뮬레이션합니다. 수익률, MDD(최대 낙폭), 샤프비율 등 성과 지표를 계산합니다 |
| `src/tools/insider_tracker.py` (내부자 거래 추적기) | 기업 임원이나 대주주가 자사 주식을 사고 팔았는지 추적합니다. 임원들이 대량 매수하면 호재 신호, 대량 매도하면 주의 신호가 될 수 있습니다 |
| `src/tools/dividend_calendar.py` (배당 캘린더) | 언제 어떤 기업이 배당금(주주에게 나눠주는 이익)을 주는지, 배당수익률이 높은 종목은 뭔지 조회합니다 |

### 수정한 파일

| 파일 | 설명 |
|------|------|
| `src/tools/pool.py` (도구 등록소) | 5개 신규 도구를 시스템에 등록. 이 파일에 등록해야 에이전트들이 도구를 사용할 수 있음 |
| `config/tools.yaml` (도구 설정 파일) | 5개 도구의 이름, 설명, 사용할 AI 모델 설정 추가 |
| `config/agents.yaml` (에이전트 설정 파일) | 투자분석처 에이전트 4명에게 신규 도구 사용 권한 배정 |
