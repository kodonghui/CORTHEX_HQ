### 8가지 수정 내역

#### 1. 예산 카드 "오늘 사용" = $0.0000
- **문제**: 전력분석에서 $1.2069 사용됐는데, 예산 카드에는 $0.0000 표시
- **근본 원인 2가지**:
  - 프론트엔드가 `budget.today_spent`를 읽는데, API는 `daily_used`를 반환 → `undefined` → 0
  - `get_today_cost()`가 `tasks` 테이블만 조회. 실제 비용은 `agent_calls` 테이블에 기록됨
- **수정**:
  - `db.py`: `get_today_cost()`에서 `tasks` + `agent_calls` 양쪽 비용 합산
  - `arm_server.py`: API 응답에 `today_spent` 필드 추가

#### 2. 슬래시 팝업 너무 투명
- **문제**: 슬래시 명령어 팝업 배경이 반투명이라 글자 읽기 어려움
- **원인**: `glass` 클래스 = 55% 투명 배경
- **수정**: `glass` 제거 → 불투명 배경(`bg-[rgb(var(--hq-surface))]`)으로 교체

#### 3. 슬래시 팝업이 대화 뒤에 숨김
- **문제**: 대화가 있는 상태에서 `/` 입력하면 팝업이 채팅 메시지 뒤에 숨겨짐
- **원인**: Input Bar에 z-index가 없어서 채팅 영역과 같은 레이어
- **수정**: Input Bar에 `relative z-50` 추가, 팝업을 `z-[999]`로 최상위

#### 4. 도구 수 111개 → 154개
- **문제**: `/명령어` 입력 시 "111개 도구 상태 점검"으로 표시
- **원인**: `arm_server.py`에 "111개"가 글자로 직접 적혀있었음 (하드코딩)
- **수정**: `len(_TOOLS_LIST)` (실제 도구 수)로 자동 계산

#### 5. 전력분석에 스페셜리스트 없음
- **문제**: 전력분석 탭에 비서실장과 CMO만 표시. 전문가급 에이전트 안 나옴
- **원인**: `tasks` 테이블에는 처장급만 기록됨. `agent_calls` 테이블 데이터를 가져오지만 사용하지 않음
- **수정**: `agent_calls` 데이터를 전력분석 목록에 병합. 이미 있는 에이전트는 수치 합산, 없는 에이전트(스페셜리스트)는 새로 추가

#### 6. 작업일지 디자인 개선
- **문제**: 카드에 체크박스, 별, 상태뱃지, 시간, 태그, 명령어, 요약, 화살표가 너무 많아 조잡
- **수정**:
  - 좌측 상태 컬러바 추가 (완료=초록, 실패=빨강, 진행중=노랑)
  - 반투명 `glass` → 불투명 `bg-hq-panel` 배경
  - 체크박스·별·💬·🗑️를 우측 상단에 모아 정렬
  - 비용/시간을 상단 우측에 배치
  - "▼ 눌러서 상세 보기" 텍스트 제거 (호버 효과로 대체)

#### 7. 작업일지 완료 v 버튼 위치
- **문제**: 시간/비용/버튼이 카드 오른쪽 끝에 떨어져 있음
- **수정**: #6 디자인 개선에 통합 — 모든 액션 버튼을 상단 우측에 모아 깔끔하게 배치

#### 8. CEO > 화살표 스타일 차이
- **문제**: CEO만 SVG 꺾은선 화살표(`>`), 부서는 유니코드 삼각형(`▶`)
- **수정**: CEO도 동일한 유니코드 삼각형(`▶`)으로 통일
