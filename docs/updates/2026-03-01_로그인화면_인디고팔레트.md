## 빌드 #741 — 로그인 화면 풀스크린 재설계

### 배경
기존 로그인 오버레이가 bootstrap_mode 우회로 스킵되는 버그 존재.
"접속 전에 로그인 화면이 보여야 한다"는 대표님 요구 → 전면 재설계.

### 변경 내용
**`web/static/js/corthex-app.js`**
- `checkAuth()` 수정: 토큰 없으면 bootstrap_mode 여부와 무관하게 항상 `auth.showLogin = true`
- `doLogout()` 수정: bootstrap mode 예외 제거 → 항상 로그인 화면으로

**`web/static/css/corthex-styles.css`**
- `.hq-login-grid`: 에메랄드 그리드 배경 + opacity 애니메이션
- `.hq-login-scanline`: 스캔라인 효과
- `.hq-login-logo`: 에메랄드↔청록 shimmer 그라데이션 애니메이션 (`loginLogoShimmer`)
- `.hq-login-card`: backdrop-blur + 에메랄드 보더 + glow shadow
- `.hq-login-input`: 모노스페이스, letter-spacing 0.15em
- `.hq-login-btn`: hover shimmer 슬라이드 효과 (::before)
- `.hq-login-corner`: Geist Mono, tracking-[0.15em]

**`web/templates/index.html`**
- 기존 로그인 오버레이 → 풀스크린 (`fixed inset-0 z-[60]`) 전면 교체
- 코너 텍스트: `SYS://CORTHEX-HQ/AUTH`, `█ SECURE`, `BUILD #740 · v4.00.004`
- 에메랄드 shimmer CORTHEX 로고 + `● SYSTEM ONLINE` 상태바
- 역할 탭: `▸ 대표님` / `▸ 누나`
- Bootstrap mode 시 초기 비밀번호 힌트 표시
