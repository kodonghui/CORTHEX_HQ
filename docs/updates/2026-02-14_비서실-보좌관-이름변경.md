# 비서실 보좌관 이름 변경

## 작업 날짜
2026-02-14

## 작업 브랜치
claude/secretary-names-brainstorm-VXANr

## 변경 사항 요약

비서실 소속 보좌관 3명의 이름을 CEO 요청으로 변경했습니다.

### 이름 변경 내역

| 역할 | 이전 이름 | 새 이름 | 하는 일 |
|------|----------|---------|---------|
| 보고 요약 담당 | 브리핑 보좌관 | **총괄 보좌관** | 긴 보고서를 3~5줄로 핵심만 압축 |
| 일정 추적 담당 | 작전 보좌관 | **진행 보좌관** | 날짜, 기한, 후속조치 추적 |
| 정보 중계 담당 | 부서 보좌관 | **소통 보좌관** | 부서 간 관련 정보 전달 |

> 비서실장 이름은 그대로 "비서실장" 유지

### 수정한 파일 목록

| 파일 | 설명 | 뭘 바꿨나 |
|------|------|----------|
| `config/agents.yaml` (에이전트 설정 파일) | 한국어 이름 + 영어 이름 변경 | name_ko, name 필드 |
| `web/templates/index.html` (홈페이지 화면) | 화면에 표시되는 이름 변경 | agentNames 매핑 |
| `souls/agents/report_worker.md` (총괄 보좌관 설정) | 제목, 노션 보고용 Agent 이름 변경 | 문서 제목 + Agent 값 |
| `souls/agents/schedule_worker.md` (진행 보좌관 설정) | 제목, 노션 보고용 Agent 이름 변경 | 문서 제목 + Agent 값 |
| `souls/agents/relay_worker.md` (소통 보좌관 설정) | 제목, 노션 보고용 Agent 이름 변경 | 문서 제목 + Agent 값 |
| `souls/agents/chief_of_staff.md` (비서실장 설정) | 하위 보좌관 이름 표 변경 | Worker 테이블 |
