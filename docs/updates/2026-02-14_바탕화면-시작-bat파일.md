## 변경 사항 요약

### 새로 만든 파일
- `CORTHEX_바탕화면_시작.bat` (바탕화면에서 더블클릭으로 CORTHEX HQ를 바로 시작하는 파일)

### 이 파일이 하는 일
기존에는 두 가지 파일을 따로 실행해야 했습니다:
1. `setup.bat` — 처음 설치할 때 (Python 확인, 필요한 것들 다운로드)
2. `run.bat` — 서버 켤 때 (매번 실행)

**새로 만든 `CORTHEX_바탕화면_시작.bat`**은 이 두 가지를 합쳐서:
- 처음 실행하면 → 자동 설치 + 서버 시작
- 두 번째부터는 → 바로 서버 시작

바탕화면에 복사해두고 더블클릭만 하면 됩니다.

### 사용 방법
1. `CORTHEX_바탕화면_시작.bat` 파일을 바탕화면에 복사
2. 파일을 메모장으로 열어서 `CORTHEX_DIR=` 뒤의 경로를 본인 컴퓨터의 CORTHEX_HQ 폴더 경로로 수정
   - 기본값: `C:\Users\(내 이름)\Documents\GitHub\CORTHEX_HQ`
3. 저장 후 더블클릭하면 끝!
