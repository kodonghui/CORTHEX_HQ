## 변경 사항 요약

### 1. `.env.local` 보안 강화
- **뭘 했는지**: `.env.local` 파일(API 키, 비밀번호 등이 담긴 설정 파일)을 GitHub에서 제거
- **왜 했는지**: 이 파일이 GitHub에 올라가 있어서 `git pull`(파일 받기)할 때마다 CEO님의 실제 설정이 초기화되는 문제가 있었음
- **결과**: 이제 `git pull`해도 `.env.local`이 절대 건드려지지 않음. CEO님 설정이 영구 보존됨

### 2. 주기적 pending(미승인) 작업 알림 타이머 추가
- **뭘 했는지**: 서버가 30분마다 "승인 대기 중인 SNS 글이 있나?" 자동 확인하는 기능 추가
- **왜 했는지**: 이전에는 작업이 완료된 직후에만 확인했음. CEO님이 승인을 깜빡해도 아무 알림이 없었음
- **결과**: 30분마다 자동 확인 → 미승인 글이 있으면 텔레그램으로 CEO님께 알림 전송
