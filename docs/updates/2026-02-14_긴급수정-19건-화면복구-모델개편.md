## 변경 사항 요약

### 무엇이 문제였나?

1. **화면이 아예 안 보였습니다** — 색상 시스템을 CSS 변수 방식으로 바꿨는데, 이 방식이 Tailwind CDN(인터넷에서 불러오는 디자인 도구)에서 작동하지 않아서 모든 글자가 투명해져 버렸습니다.
2. **모델 선택 화면에 "undefined"** — 검수 모델과 에이전트 모델 선택할 때 가격이 "undefined"로 나왔습니다.
3. **추론 정도 설정 불가** — Claude Sonnet/Opus 같은 모델을 선택해도 추론 강도를 조절할 수 없었습니다.
4. **최신 GPT 모델이 없었습니다** — GPT-5.1, 5.2 등 최신 모델이 목록에 없었습니다.
5. **출판기록 부서 에이전트가 1명** — 원래 4명이어야 하는데 1명만 있었고, 그마저도 ID가 틀려서 "not found" 에러가 났습니다.

### 수정한 내용 (총 19건, 4그룹)

#### 화면 복구 (5건)
| 뭘 고쳤나 | 설명 |
|-----------|------|
| Tailwind 색상 복원 | CSS 변수 방식 → 원래 hex 색상코드(#f1f5f9 등)로 되돌림 |
| CSS 변수 삭제 | 쓸모없어진 `:root` 색상 변수 정의 삭제 |
| glass 효과 복원 | 반투명 유리 효과 배경이 투명해진 것 → rgba 색상으로 복원 |
| gradient-border 복원 | 그라데이션 테두리 배경 복원 |
| 라이트모드 강화 | 밝은 모드에서 배경, 텍스트, 테두리, 카드 등 15가지 색상 오버라이드(덮어쓰기) 추가 |

#### 모델 관련 (6건)
| 뭘 고쳤나 | 설명 |
|-----------|------|
| GPT-5 최신 모델 추가 | models.yaml에 GPT-5-Nano, 5.1, 5.1-Codex, 5.2, 5.2-Codex, 5.2-Pro 추가 (총 11개 모델) |
| 모델 순서 정렬 | 비싼 모델(Opus, GPT-5.2 Pro)이 위에, 싼 모델(Nano, Haiku)이 아래에 오도록 |
| 검수 모델 undefined 수정 | 가격이 "undefined"로 나오던 것 → "$5.00/$25.00" 형식으로 정상 표시 |
| 추론 fallback 업데이트 | 모델 10종 모두의 추론 단계(low/medium/high/xhigh)를 하드코딩으로 보장 |
| 부서별 검수 모델 설정 | 각 부서별로 검수에 다른 AI 모델과 추론 강도를 따로 설정 가능 |
| 모델 이름 개선 | "claude-opus-4-6" → "Claude Opus 4.6 ($5.00/$25.00)" 읽기 좋은 형식 |
