# 29명 에이전트 소울 업그레이드 (전체 완료)

## 작업 날짜

## 작업 브랜치
claude/soul-upgrade-batch2-16to29

## 변경 사항 요약

### 뭘 했나?
29명의 AI 에이전트 소울 파일에 **성격, 말투, 금지 사항, 협업 규칙** 4가지 섹션을 추가했습니다.
쉽게 말하면, 각 AI 직원에게 "너는 어떤 성격이고, 어떤 말투를 쓰고, 뭘 하면 안 되고, 누구와 어떻게 일하는지"를 설정해준 것입니다.

### 왜 했나?
기존에는 에이전트들이 "뭘 하는 사람인지(역할)"와 "어떤 도구를 쓰는지"만 정해져 있었습니다.
하지만 **성격과 말투가 없으면** AI가 모두 같은 톤으로 말하게 되어, 29명이 구분이 안 됩니다.
이번 업그레이드로 각 에이전트가 **자기만의 성격과 말투**로 답변하게 됩니다.

### 각 파일에 추가된 4가지 섹션

1. **## 성격** — 에이전트의 고유 성격 특성 4가지 (예: 비서실장은 "완벽주의 스케줄러", CIO는 "감정 제로 투자 전략가")
2. **## 말투** — 대화 스타일 설명 + 실제 대화 예시 4개 (예: 특허 전문가는 "법률 문서체 존댓말", 커뮤니티 전문가는 "친근하고 따뜻한 존댓말")
3. **## 금지 사항** — 절대 하면 안 되는 행동 4~5개 (예: CMO는 "CEO 승인 없이 외부 콘텐츠 발행 금지")
4. **## 협업 규칙** — 상관/부하/동료 관계 + 보고 라인 (예: 종목분석 전문가의 상관은 CIO, 동료는 시황분석/기술적분석/리스크관리)

### 건드리지 않은 것
- 기존 **## 업무 원칙** / **## 업무 철학** 섹션 — 한 글자도 수정 안 함
- 기존 **## 내가 쓰는 도구** 섹션 — 한 글자도 수정 안 함
- **## 📤 노션 보고 의무** 섹션 — 한 글자도 수정 안 함
