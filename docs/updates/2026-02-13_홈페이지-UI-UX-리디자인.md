## 변경 사항 요약

### 1. 브랜드 시스템 구축
- CORTHEX 전용 로고(육각형 신경망 모양)를 SVG로 직접 만듦
- 사이언(cyan) → 파랑 → 보라 그라디언트 색상 체계 적용
- 유리처럼 반투명한 카드 디자인(글래스모피즘) 도입

### 2. 웰컴 화면 리디자인
- 처음 접속하면 로고 + 브랜드명 + 간단 소개가 나오는 웰컴 화면 추가
- 명령 입력하면 자연스럽게 채팅 화면으로 전환

### 3. 사이드바 리팩토링
- "조직도"와 "활동 로그"를 접었다 펼 수 있는 토글 방식으로 변경
- 에이전트마다 이름 이니셜 + 본부별 색상 아바타 표시
- 본부별 배지 색상 구분 (비서실=사이언, 투자=보라, 리트마스터=주황 등)

### 4. 채팅 UX 업그레이드
- AI 응답을 "결과 헤더"와 "본문"으로 깔끔하게 분리
- 입력창에 포커스하면 은은하게 빛나는 효과 추가
- 마크다운 렌더링 스타일 개선 (코드 블록, 표 등)

### 5. 마이크로 인터랙션
- 카드에 마우스 올리면 살짝 떠오르는 효과 (hover-lift)
- 에이전트 카드에 빛나는 테두리 효과 (hover-glow)
- 새 요소가 나타날 때 아래에서 위로 올라오는 애니메이션 (fade-in-up)

### 6. 한국어 레이블 전면 복원
- 영어로 되어 있던 모든 UI 텍스트를 한국어로 변경
- 대기/처리중/오류/연결됨/보내기/조직도/활동 로그 등 약 20곳 이상

### 7. 활동 로그 시간 KST 적용
- 로그에 뜨는 시간을 한국 시간(KST, UTC+9)으로 표시하도록 변경

### 8. CLAUDE.md에 UI/UX 규칙 추가
- 모든 UI 한국어 작성, KST 시간대, 디자인 시스템 규칙 문서화
