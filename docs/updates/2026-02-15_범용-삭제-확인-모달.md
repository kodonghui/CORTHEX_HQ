## 변경 사항 요약

### 뭘 바꿨나?
예약 탭과 워크플로우 탭에서 "삭제" 버튼을 눌렀을 때 나오는 **삭제 확인 창**을 새로 만들었습니다.

- **이전**: 예약 탭에만 삭제 확인 창이 있었고, 워크플로우는 확인 없이 바로 삭제됨
- **이후**: 예약/워크플로우 둘 다 같은 예쁜 확인 창으로 통합

### 디자인 변경점
| 항목 | 이전 | 이후 |
|------|------|------|
| 아이콘 크기 | 64px | 80px (더 크고 빨간 빛남 효과) |
| 삭제 대상 이름 | 일반 텍스트 | 빨간 박스로 하이라이트 |
| 버튼 배치 | 가로 나란히 | 세로 전체 너비 (삭제 위, 취소 아래) |
| 상단 효과 | 없음 | 빨간 그라데이션 라인 |
| 모서리 | 둥글게 (16px) | 더 둥글게 (24px) |
| 워크플로우 | 확인 없이 바로 삭제 | 동일한 확인 창 표시 |

### 수정한 파일
- `web/templates/index.html` (홈페이지 메인 화면)
  - 앱 데이터에 `confirmDelete` 범용 상태 추가
  - 기존 예약 전용 삭제 모달 → 범용 모달로 교체
  - 예약 삭제 버튼 → 범용 모달 연결
  - 워크플로우 삭제 버튼 → 범용 모달 연결
