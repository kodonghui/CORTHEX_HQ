## 변경 사항 요약

### 1단계: 뉴트럴 다크 + 라이트모드 수리 (이전 작업)

- 남색 팔레트 → 무채색 팔레트로 교체
- 194개 색상 클래스를 CSS 변수 기반으로 전환
- 라이트모드 버그(색상 안 바뀜) 완전 수정

### 2단계: Glass + Aurora Mix (현재 작업)

**CEO 피드백**: "너무 어둡고 너무 밝다", "보내기 버튼 글자가 안 보인다"

#### 해결한 것들:

**① 따뜻한 톤 팔레트로 교체**
| 영역 | 이전 (차가운 무채색) | 변경 후 (따뜻한 톤) |
|------|-------------------|------------------|
| 배경 | `rgb(13 17 23)` (차가운 검정) | `rgb(15 14 22)` (따뜻한 보라빛 검정) |
| 카드 | `rgb(22 27 34)` (차가운 회색) | `rgb(26 24 36)` (따뜻한 보라빛 회색) |
| 테두리 | `rgb(48 54 65)` (차가운 회색) | `rgb(58 55 75)` (따뜻한 보라빛 회색) |
| 글자 | `rgb(237 242 247)` (차가운 흰색) | `rgb(235 232 245)` (따뜻한 라벤더 흰색) |

> 차가운 느낌이 사라지고, 전체적으로 **따뜻하고 고급스러운 느낌**이 됨

**② 배경을 그라데이션으로 교체**
- 이전: 단색 검정 (`rgb(10 12 18)`)
- 이후: **오로라 그라데이션** — 왼쪽 위 보라빛 → 오른쪽 청록빛으로 은은하게 흐르는 효과
- 배경 위에 **보라색 글로우**와 **청록색 글로우**가 은은하게 떠있음 (body::before, ::after)
- 라이트모드도 따뜻한 연보라 그라데이션

**③ 유리 효과 (Glassmorphism) 적용**
- 모든 카드, 사이드바, 헤더, 입력창에 **반투명 배경 + backdrop-blur** 적용
- 뒤에 있는 오로라 그라데이션이 유리를 통해 살짝 비치는 효과
- 테두리는 `rgb(255 255 255 / 0.06)` — 빛이 비치는 듯한 미세한 밝은 선
- hover 시 보라색 빛이 은은하게 퍼지는 효과

**④ 보내기 버튼 글자 수정**
- 문제: `body { color: !important }` 규칙이 버튼의 `text-white` 클래스를 덮어써서 글자가 안 보였음
- 해결: `.text-white { color: #ffffff !important; }` 추가 — 흰색 글자가 확실히 보이게

## 기술적 변경 상세

1. **CSS 변수 팔레트** — 무채색에서 따뜻한 보라 톤으로 교체 (보라기 가미)
2. **body 배경** — `linear-gradient(135deg, #0F0E16, #1A1428, #0E1520, #12101E)` + `background-attachment: fixed`
3. **오로라 글로우** — `body::before`(보라 radial-gradient) + `body::after`(청록 radial-gradient) — `pointer-events: none`으로 클릭 안 막음
4. **Glass 효과** — `backdrop-filter: blur(20px) saturate(1.4)`, 배경 투명도 0.45~0.55, 테두리 `rgb(255 255 255 / 0.06)`
5. **사이드바** — `blur(24px) saturate(1.5)` — 더 깊은 유리 효과
6. **입력창** — `blur(8px)` + 반투명 배경
7. **hover-glow** — 보라색 + 파란색 그림자 복합
8. **text-white 보호** — `.text-white { color: #ffffff !important; }` 추가
