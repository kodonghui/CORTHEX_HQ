### 기술 상세

**브레이크포인트** (Tailwind 기본값):
- 📱 모바일: <768px (하단 네비 + 패딩 축소)
- 📱 소형 모바일: <480px (1열 그리드 + 더보기 3열)
- 💻 태블릿: 768-1023px (데스크탑 탭바 + 약간 축소 패딩)
- 🖥️ 데스크탑: ≥1024px (변화 없음)

**CSS 전략**: 마커 클래스 + @media 쿼리
- `desktop-tab-bar`, `header-metrics`, `header-view-toggle`, `header-build-badge`, `header-conn-text` → 모바일 숨김
- `tab-content-padded` → 모바일 패딩 축소
- `mobile-bottom-nav`, `mobile-more-sheet` → 모바일 전용 표시

**JS 변경**:
- `mobileMoreOpen` 상태 추가
- `window.resize` 핸들러: 데스크탑↔모바일 전환 시 사이드바 자동 열림/닫힘
- `switchTab()`: 모바일에서 탭 전환 시 사이드바 자동 닫힘
