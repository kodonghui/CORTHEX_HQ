## 1️⃣ GitHub API - 코드 저장소 자동화

### 🔍 GitHub API란?
GitHub(깃허브)에 저장된 코드를 자동으로 관리하고 제어할 수 있는 도구입니다.
개발자가 직접 클릭하지 않아도 프로그램이 알아서 이슈를 만들고, 코드 리뷰를 하고, 자동으로 배포까지 할 수 있게 해줍니다.

### 📦 두 가지 버전
1. **REST API v3** - 전통적인 방식. 필요한 정보를 하나씩 요청해야 함
2. **GraphQL API v4** - 최신 방식. 한 번에 여러 정보를 요청할 수 있어서 더 빠르고 효율적

### ✅ 할 수 있는 일
- **이슈/PR 관리** - 버그 신고나 기능 요청을 자동으로 정리
- **코드 리뷰 자동화** - 코드에 문제가 있는지 자동으로 검사
- **GitHub Actions 연동** - 코드를 올리면 자동으로 테스트하고 배포
- **자동 커밋/푸시** - 정해진 시간에 자동으로 코드 저장

### 💰 무료 범위
- **공개 저장소(Public)**: 완전 무료, 제한 없음
- **비공개 저장소(Private)**: 한 달에 2,000분 무료 자동화 시간 제공
- API 호출 횟수 제한 있음 (인증하면 더 많이 사용 가능)

### 📚 참고 자료
- [GitHub REST API vs GraphQL API 비교](https://docs.github.com/en/rest/about-the-rest-api/comparing-githubs-rest-api-and-graphql-api)
- [GitHub Actions 공식 문서](https://github.com/features/actions)

## 2️⃣ 코드 품질 및 보안 검사 도구

### 🛡️ SonarQube/SonarCloud - 코드 품질 검사

#### 🔍 SonarQube란?
코드에 숨어있는 버그, 보안 취약점, 나쁜 코딩 습관을 자동으로 찾아주는 도구입니다.
코드를 올리면 자동으로 검사해서 "이 부분은 나중에 문제가 될 수 있어요"라고 알려줍니다.

#### ✅ Python 지원
- Python 코드 전용 검사 규칙 수백 개 제공
- 버그, 보안 취약점, 코드 냄새(나쁜 습관) 자동 탐지
- Jenkins, GitHub Actions, GitLab CI 등과 쉽게 연동

#### 💰 무료 범위
- **개인 개발자용 무료 티어**: 코드 5만 줄까지 무료
  - 비공개 프로젝트도 무료로 사용 가능
  - 자동 코드 리뷰 기능 포함
- **유료 팀 플랜**: 월 32달러부터 (약 4만 원)

#### 📊 언제 사용하나요?
- 작은 스타트업이나 개인 프로젝트 (5만 줄 미만) → **무료로 충분**
- 중대형 프로젝트 (10만 줄 이상) → 유료 필요

#### 📚 참고 자료
- [SonarQube Cloud 무료 가입](https://www.sonarsource.com/products/sonarqube/cloud/signup-free/)
- [Python 코드 품질 분석](https://www.sonarsource.com/knowledge/languages/python/)

### 🔒 Snyk - 보안 취약점 검사

#### 🔍 Snyk란?
코드와 라이브러리(외부에서 가져온 코드)에 보안 구멍이 있는지 자동으로 찾아주는 도구입니다.
"이 라이브러리 버전은 해킹 위험이 있으니 업데이트하세요"라고 알려줍니다.

#### ✅ 주요 기능
- 코드 스캔 (내가 작성한 코드의 보안 검사)
- 의존성 스캔 (외부 라이브러리의 보안 검사)
- AI 기반 실시간 피드백
- IDE(개발 도구)에 바로 설치해서 사용 가능

#### 💰 무료 범위
- 한 달에 100~200회 검사 무료
- 공개 저장소 무제한
- 기본 취약점 스캔 포함
- GitHub와 매우 쉽게 연동

#### 📚 참고 자료
- [Snyk 공식 사이트](https://snyk.io/)
- [2026 Snyk vs Semgrep 비교](https://sanj.dev/post/ai-code-security-tools-comparison)

### 🔓 Semgrep - 오픈소스 보안 검사 (완전 무료!)

#### 🔍 Semgrep이란?
완전히 무료로 사용할 수 있는 오픈소스 보안 검사 도구입니다.
Facebook(메타)에서 만들어서 신뢰도가 높고, 속도가 빠르며, 잘못된 경고가 적습니다.

#### ✅ 주요 기능
- **완전 무료 오픈소스** - 기업도 무료로 사용 가능
- 30개 이상 프로그래밍 언어 지원 (Python 포함)
- IDE, pre-commit(커밋 전 자동 검사), CI/CD 모두 연동 가능
- 빠른 속도 + 정확한 검사

#### 💰 무료 범위
- 오픈소스 버전: **완전 무료** (기업도 사용 가능)
- 유료 팀 플랜: 월 40달러/개발자 (팀 관리 기능 추가)

#### 🆚 Snyk vs Semgrep 비교
| 항목 | Snyk | Semgrep |
|------|------|---------|
| **가격** | 한 달 100회 무료 | 완전 무료 (오픈소스) |
| **검사 방식** | AI 기반 | 패턴 매칭 + 의미 분석 |
| **속도** | 보통 | 매우 빠름 |
| **잘못된 경고** | 좀 있음 | 적음 |

#### 📚 참고 자료
- [Semgrep GitHub](https://github.com/semgrep/semgrep)
- [Semgrep 공식 사이트](https://semgrep.dev/)

## 3️⃣ 모니터링 및 에러 추적 도구

### 🚨 Sentry - 에러 추적 및 성능 모니터링

#### 🔍 Sentry란?
서비스가 운영 중일 때 발생하는 에러를 실시간으로 잡아서 알려주는 도구입니다.
"지금 로그인 기능에서 에러가 100번 발생했어요. 이 코드 때문입니다"라고 정확히 알려줍니다.

#### ✅ 주요 기능
- **에러 추적**: 어디서 에러가 났는지 실시간 알림
- **성능 모니터링**: 페이지 로딩 속도, 응답 시간 측정
- **로그 관리**: 5GB 로그 저장 제공
- **업타임 모니터**: 서비스가 잘 돌아가는지 확인

#### 💰 무료 범위
- **개발자 플랜**: 한 달에 5,000개 이벤트 무료
  - 에러 알림
  - 5GB 로그 저장
  - 1개 크론 모니터(정기 작업 감시)
  - 1개 업타임 모니터(서비스 상태 확인)
- **유료 플랜**: 연 500~10,000달러 (팀 규모별)

#### 🎯 언제 사용하나요?
- 작은 프로젝트나 초기 스타트업 → **무료로 충분**
- 사용자가 많은 서비스 → 유료 필요

## 4️⃣ CI/CD 자동화 - 배포 자동화

### ⚡ GitHub Actions - GitHub 내장 자동화

#### 🔍 GitHub Actions란?
코드를 GitHub에 올리면 자동으로 테스트하고, 빌드하고, 배포까지 해주는 도구입니다.
개발자가 "배포" 버튼을 누르지 않아도 자동으로 진행됩니다.

#### ✅ 주요 기능
- 코드 푸시하면 자동 테스트
- 테스트 통과하면 자동 배포
- 스케줄링 (매일 자정에 자동 실행 등)
- 30개 이상 언어 지원

#### 💰 무료 범위
- **공개 저장소**: 완전 무료, 무제한
- **비공개 저장소**: 한 달 2,000분 무료
- Linux 서버 사용 시 가장 저렴

#### 🆚 2026년 가격 변화
- **좋은 소식**: 호스팅 서버 가격 39% 인하 (2026년 1월부터)
- **나쁜 소식**: 자체 서버 사용 시 분당 $0.002 부과 (2026년 3월부터)

#### 📁 설정 방법
`.github/workflows/` 폴더에 YAML 파일 작성하면 끝!

```yaml
name: 자동 배포
on:
  push:
    branches: [ main ]
jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      - name: 테스트 실행
        run: npm test
      - name: 배포
        run: npm run deploy
```

### 🚀 배포 플랫폼 - Vercel, Railway, Render

#### 🔍 배포 플랫폼이란?
만든 웹사이트나 앱을 인터넷에 올려서 사람들이 쓸 수 있게 해주는 서비스입니다.
복잡한 서버 설정 없이 GitHub와 연결만 하면 자동으로 배포됩니다.

---

#### 🌐 Vercel - 프론트엔드 특화

**✅ 장점**
- Next.js 만든 회사라서 최적화 완벽
- 무료 호비 플랜 제공
- 자동 배포 + 전 세계 빠른 속도
- 설정 거의 불필요 (제로 설정)

**💰 무료 범위**
- 개인 프로젝트 무료
- 상업용은 제한 있음
- 빌드 시간, 대역폭 제한

**⚠️ 주의사항**
- 프론트엔드(화면)는 좋지만 백엔드(서버)는 약함
- 데이터베이스나 복잡한 서버 로직은 다른 곳에서 해야 함

---

#### 🚂 Railway - 풀스택 프로젝트에 최적

**✅ 장점**
- 백엔드 + 데이터베이스까지 모두 지원
- 자동 감지해서 설정 없이 배포
- 2분 안에 GitHub 연결해서 URL 받기
- Dockerfile 없이도 자동 빌드

**💰 무료 범위**
- 30일 무료 체험 ($5 크레딧)
- **영구 무료 없음** → 체험 후 유료 필요
- Hobby 플랜: 월 $5 (대부분 프로젝트 충분)
- Pro 플랜: 월 $20

**🎯 언제 사용하나요?**
- Vercel(화면) + Railway(서버 + DB) 조합 추천
- 작은 스타트업 월 $5면 충분

---

#### 🎨 Render - 중간 옵션

**✅ 장점**
- 정적 사이트 무료
- 월 $7부터 웹 서비스 제공
- SSL, CDN, Git 자동 배포 포함
- Dockerfile 또는 자동 감지

**💰 무료 범위**
- 정적 사이트 영구 무료
- 웹 서비스는 제한된 시간만 무료
- 유료 플랜: 월 $19부터

---

#### 📊 3가지 플랫폼 비교

| 항목 | Vercel | Railway | Render |
|------|--------|---------|--------|
| **무료 범위** | 개인 프로젝트 무료 | 30일 체험 ($5) | 정적 사이트 무료 |
| **특화 분야** | 프론트엔드 (Next.js) | 풀스택 + DB | 중간 옵션 |
| **백엔드 지원** | 약함 | 강함 | 중간 |
| **데이터베이스** | 외부 연동 필요 | 내장 지원 | 제한적 지원 |
| **유료 가격** | 상업용만 유료 | 월 $5 | 월 $19 |

#### 💡 추천 조합
- **프론트엔드만**: Vercel 무료
- **풀스택 프로젝트**: Vercel(화면) + Railway(서버+DB) → 월 $5
- **초보자**: Render 정적 사이트 무료부터 시작
