## 1. 특허 검색 API

### 1.1 KIPRIS (키프리스) - 한국 특허정보 검색 서비스 ⭐️ 추천

**무엇을 하는 서비스인가요?**
- 한국특허정보원에서 운영하는 공식 특허 검색 서비스
- 한국의 특허, 실용신안, 디자인, 상표 정보를 검색할 수 있음
- 해외 특허 정보도 일부 제공

**비용**
- ✅ **무료** (공공기관, 중소기업, 연구소 등 누구나 사용 가능)

**주요 기능**
- Open API 방식으로 프로그램과 연동 가능
- 약 78만 건 이상의 특허 데이터 제공
- 실시간 특허 상태 확인 가능

**어떻게 사용하나요?**
1. KIPRIS Plus 웹사이트 ([https://plus.kipris.or.kr](https://plus.kipris.or.kr))에서 회원가입
2. API 신청 후 인증키 발급받기
3. Python 등 프로그래밍 언어로 연동

**문의**
- 전화: 02-6915-1553 (평일 09:00~18:00)

**Python 연동**
- 가능 (REST API 방식)
- 공식 개발 가이드 제공 (KIPRISPlus Open API 이용 가이드)

---

### 1.2 Google Patents (구글 특허 검색)

**무엇을 하는 서비스인가요?**
- 구글에서 제공하는 전 세계 특허 검색 서비스
- 미국, 유럽, 중국, 일본 등 100개 이상 특허청 데이터 포함
- 한국 특허도 검색 가능

**비용**
- ⚠️ **공식 API는 폐지됨** (2016년 이후)
- 대안: Google Custom Search API 또는 서드파티 스크래핑 도구 사용 (유료)

**한국 특허 검색**
- ✅ 가능 (KIPRIS 데이터와 연동됨)

**Python 연동**
- 공식 API는 없지만, SerpApi 같은 서드파티 서비스 이용 가능 (유료)

---

### 1.3 USPTO API (미국 특허청 API)

**무엇을 하는 서비스인가요?**
- 미국 특허청(USPTO)의 공식 특허 데이터베이스
- 미국 특허 및 상표 정보 제공

**비용**
- ✅ **무료** (PatentsView API 제공)

**한국 특허 검색**
- ❌ 불가능 (미국 특허만 제공)

**Python 연동**
- ✅ 가능 (JSON API 방식)
- PatentsView API 공식 문서 제공

**용도**
- 미국 시장 진출 시 특허 조사용
- 글로벌 특허 전략 수립 시 활용

---

### 1.4 WIPO Patentscope (국제특허 검색)

**무엇을 하는 서비스인가요?**
- 세계지식재산기구(WIPO)에서 운영하는 국제 특허 검색 서비스
- PCT (국제특허협력조약) 출원 정보 240만 건 이상
- 전 세계 특허 문서 9,900만 건 이상

**비용**
- ✅ **무료** (API Catalog 제공)

**한국 특허 검색**
- ✅ 가능 (한국 특허청 데이터 포함)

**Python 연동**
- ✅ 가능 (공식 API Catalog: [https://apicatalog.wipo.int](https://apicatalog.wipo.int))

**특징**
- 번역 기능 제공 (영어, 프랑스어, 스페인어 등)
- 최대 10,000건까지 한 번에 다운로드 가능

## 2. 법령/판례 검색 API

### 2.1 국가법령정보센터 Open API ⭐️ 추천

**무엇을 하는 서비스인가요?**
- 법제처(정부 법령 담당 기관)에서 운영하는 공식 법령 검색 서비스
- 대한민국의 모든 법령, 행정규칙, 지방자치법규 검색 가능

**비용**
- ✅ **무료** (개발 계정: 하루 10,000건 요청 가능)
- 더 많이 사용하려면 사용 사례 등록 후 트래픽 증량 신청 가능

**주요 기능**
- 법령 목록 조회
- 행정규칙 정보 조회
- 지방 조례 정보 조회
- 법령 해석 사례 조회

**어떻게 사용하나요?**
1. 국가법령정보 공동활용 사이트 ([https://open.law.go.kr](https://open.law.go.kr)) 회원가입
2. API 신청 및 인증키 발급
3. Python으로 REST API 호출

**최신 업데이트** (2025년 기준)
- 행정규칙: 2025년 7월 10일 업데이트
- 법령용어: 2025년 7월 17일 업데이트
- 이용통계: 2025년 7월 24일 업데이트
- ✅ 2026년 현재도 활발히 운영 중

### 2.2 판례 검색 API ⭐️ 추천

**무엇을 하는 서비스인가요?**
- 법제처에서 제공하는 대한민국 법원 판례 검색 API
- 대법원, 고등법원, 지방법원 등 모든 법원의 판례 제공

**비용**
- ✅ **무료** (공공데이터포털을 통해 제공)

**주요 기능**
1. **판례 목록 조회 API**
   - 사건번호, 사건명, 선고일자, 법원명, 재판부명, 사건 개요 제공

2. **판례 본문 조회 API**
   - 판례 전문(판결문 전체 내용) 조회
   - 사건 요약, 분류 코드, 인용 조문 정보 포함

**어떻게 사용하나요?**
- 공공데이터포털 ([https://www.data.go.kr](https://www.data.go.kr))에서 "판례" 검색
- API 활용 신청 후 인증키 발급
- Python으로 연동

**참고 사이트**
- 대법원 종합법률정보: [https://glaw.scourt.go.kr](https://glaw.scourt.go.kr)
- 사법정보공개포털: [https://portal.scourt.go.kr](https://portal.scourt.go.kr)

## 3. 저작권 관련 도구

### 3.1 한국저작권위원회 저작권 등록정보 API ⭐️ 추천

**무엇을 하는 서비스인가요?**
- 한국저작권위원회에 등록된 저작물 정보 검색 서비스
- 약 78만 건의 저작물 등록 정보 제공

**비용**
- ✅ **무료** (공공데이터포털을 통해 제공)

**주요 기능**
- 저작권 등록 정보 통합 검색
- 개별 저작물의 상세 등록 내용 조회
- 저작권자, 등록일자, 저작물 종류 등 확인 가능

**어떻게 사용하나요?**
- 공공데이터포털 ([https://www.data.go.kr/data/15106731/openapi.do](https://www.data.go.kr/data/15106731/openapi.do))에서 신청
- API 인증키 발급 후 사용

**법적 근거**
- 저작권법에 따라 한국저작권위원회가 저작권등록부를 관리
- 등록 정보는 등록공보 발행 또는 정보통신망 게시 의무 있음

**관련 사이트**
- 저작권 등록 시스템(CROS): [https://www.cros.or.kr](https://www.cros.or.kr)
- 저작권 검색(Find Copyright): [https://www.findcopyright.or.kr](https://www.findcopyright.or.kr)

## 4. 계약서/전자서명 도구

### 4.1 DocuSign (도큐사인) - 글로벌 1위 전자서명 서비스

**무엇을 하는 서비스인가요?**
- 세계에서 가장 많이 사용되는 전자계약/전자서명 플랫폼
- Fortune 100대 기업 중 99개, Fortune 500대 기업 중 450개 이상 사용

**한국 진출**
- ✅ 2024년 한국 진출 (공식 파트너: 인텔렉추얼데이터)
- 한국어 지원

**비용**
- 💰 **유료** (기업용 라이선스 필요)
- 가격은 사용 규모에 따라 달라짐

**주요 기능**
- 문서에 전자서명 추가
- 계약서 자동화 워크플로우
- API를 통한 내부 시스템 연동
- 강력한 보안 시스템

**API 제공**
- ✅ REST API 제공 (시스템 통합 가능)
- Python SDK 지원

**특징**
- 2024년 3월 USPTO(미국 특허청)가 전자서명 규정 개정 후, 특허 출원에도 DocuSign 사용 가능
- 전 세계 법적 효력 인정

### 5.2 로앤컴퍼니 - 로톡(LawTalk) / 슈퍼로이어(SuperLawyer) ⭐️ 한국 1위

**무엇을 하는 회사인가요?**
- 한국 최대 법률 플랫폼 기업
- 주요 서비스:
  - **로톡**: 종합 법률 포털 (변호사 찾기, 법률 상담)
  - **로톡 비즈**: 기업 법무 솔루션
  - **빅케이스**: AI 법률 정보 서비스
  - **슈퍼로이어**: AI 법률 비서 (2026년 주력 상품)

**슈퍼로이어 (SuperLawyer) - AI 법률 비서**
- **출시일**: 2025년 말 베타 공개, 2026년 정식 출시
- **형태**: 클라우드 기반 SaaS (웹브라우저에서 바로 사용)
- **비용**: 💰 유료 (구독 모델 예상)

**주요 기능**
1. **법률 리서치**: 판례, 법령 검색을 AI가 자동으로 수행
2. **초안 작성**: 소장, 답변서, 의견서 등 법률문서 자동 작성
3. **문서 요약**: 긴 판결문이나 계약서를 짧게 요약
4. **문서 기반 대화**: 계약서 PDF 업로드하면 AI가 분석하고 질문에 답변
5. **사건 기반 대화**: 사건 정보 입력하면 관련 판례와 법령 자동 검색

**협력사**
- 법무법인 화우와 협력하여 2026년 말까지 법률사건분석 에이전트 개발 예정

**대상 고객**
- 로펌(법무법인)
- 기업 법무팀
- 개인 변호사

**웹사이트**: [https://superlawyer.co.kr](https://superlawyer.co.kr)

## 6. 오픈소스 라이선스 검사 도구

### 6.1 FOSSA ⭐️ 글로벌 표준

**무엇을 하는 도구인가요?**
- 소프트웨어에 사용된 오픈소스 라이브러리의 라이선스를 자동으로 검사
- 라이선스 충돌(예: GPL과 MIT 혼용 시 문제) 감지
- 보안 취약점도 함께 검사

**누가 사용하나요?**
- Uber, Ford, Zendesk, Motorola 등 글로벌 기업
- Kubernetes, Webpack, Terraform, ESLint 등 7,000개 이상의 오픈소스 프로젝트

**비용**
- 💰 **유료** (SaaS 구독 모델)
- 비즈니스 트라이얼(무료 체험) 제공

**주요 기능**
1. **라이선스 스캔**: 프로젝트의 모든 오픈소스 컴포넌트와 라이선스 목록 자동 추출
2. **충돌 감지**: GPL, MIT, Apache 등 라이선스 간 충돌 경고
3. **의존성 추적**: 직접 사용한 라이브러리뿐 아니라 그 라이브러리가 또 사용한 라이브러리까지 추적
4. **실시간 알림**: 새로운 취약점이나 라이선스 문제 발생 시 즉시 알림
5. **CI/CD 통합**: GitHub, GitLab, Jenkins 등과 연동하여 코드 푸시할 때마다 자동 검사

**지원 언어/도구**
- Maven, Gradle (Java)
- npm, Yarn (JavaScript)
- Go modules, pip (Python)
- 컨테이너(Docker) 스캔도 가능

**어떻게 사용하나요?**
1. FOSSA 웹사이트 ([https://fossa.com](https://fossa.com))에서 회원가입
2. GitHub, Bitbucket, GitLab 계정 연동
3. 프로젝트 선택하면 자동으로 스캔 시작
4. CLI 도구를 CI/CD 파이프라인에 추가하면 자동화 가능

**웹사이트**: [https://fossa.com](https://fossa.com)

---

### 6.2 Snyk (스닉) ⭐️ 보안 중심 도구

**무엇을 하는 도구인가요?**
- 오픈소스 라이선스 검사 + 보안 취약점 스캔
- 개발자 도구(IDE, Git)에 직접 통합 가능

**비용**
- ✅ **무료 플랜 있음** (개인/소규모 프로젝트용)
- 💰 기업용은 유료

**주요 기능**
1. **오픈소스 취약점 스캔**: 사용 중인 라이브러리의 보안 문제 자동 감지
2. **라이선스 검사**: GPL, MIT, Apache 등 라이선스 충돌 확인
3. **자동 수정**: 취약점 발견 시 자동으로 패치 제안
4. **컨테이너 스캔**: Docker 이미지 내 취약점 검사
5. **코드 스캔**: 직접 작성한 코드의 보안 문제도 검사

**지원 언어**
- JavaScript, TypeScript, Python, Java, Go, Ruby, PHP, Scala, Swift 등
- 2026년 2월 23일에 지원 언어 확대 업데이트 예정

**어떻게 사용하나요?**
1. Snyk 웹사이트 ([https://snyk.io](https://snyk.io))에서 무료 계정 생성
2. GitHub, GitLab 등 연동
3. CLI 또는 IDE 플러그인 설치
4. 프로젝트 스캔

**특징**
- IDE(VS Code, IntelliJ 등)에 플러그인 설치하면 코딩하면서 실시간으로 취약점 알림
- CI/CD 파이프라인에 통합 가능

**웹사이트**: [https://snyk.io](https://snyk.io)

### 6.3 FOSSology (포솔로지) - 무료 오픈소스 도구

**무엇을 하는 도구인가요?**
- Linux Foundation에서 만든 오픈소스 라이선스 검증 프로그램
- 완전 무료 (오픈소스 프로젝트)

**비용**
- ✅ **완전 무료** (오픈소스)

**주요 기능**
1. 라이선스 검증
2. 저작권 정보 추출
3. 워크플로우 관리
4. 데이터베이스 및 웹 UI 제공

**어떻게 사용하나요?**
- 자체 서버에 설치하여 사용 (클라우드 SaaS가 아님)
- Docker로 쉽게 설치 가능

**장점**
- 완전 무료
- 데이터가 외부로 나가지 않음 (보안에 유리)

**단점**
- 직접 서버 관리 필요
- UI/UX가 FOSSA, Snyk보다 불편

**웹사이트**: [https://www.fossology.org](https://www.fossology.org)
**한국어 가이드**: [OpenChain 가이드](https://haksung.gitbook.io/openchainguide/appendix/tools/fossology)

### 7.2 목적별 추천

#### 🔍 특허 검색이 필요하다면
- **한국 특허**: KIPRIS API (무료) ⭐️
- **미국 특허**: USPTO API (무료)
- **전 세계 특허**: WIPO Patentscope (무료)

#### 📚 법령/판례가 필요하다면
- **법령**: 국가법령정보센터 API (무료) ⭐️
- **판례**: 법제처 판례 API (무료) + 빅케이스 AI 검색 (유료)

#### ©️ 저작권 확인이 필요하다면
- **저작권 등록 조회**: 한국저작권위원회 API (무료)
- **오픈소스 라이선스**: FOSSA (유료) 또는 Snyk (무료/유료)

#### ✍️ 계약서 작성/서명이 필요하다면
- **한국 기업**: 모두싸인 (유료, 한국 1위)
- **글로벌 기업**: DocuSign (유료, 세계 1위)
- **무료로 시작**: 유캔싸인 (무료 플랜 있음)

#### 🤖 AI 법률 서비스가 필요하다면
- **종합 법률 AI**: 슈퍼로이어 (로앤컴퍼니, 유료)
- **판례 검색**: 빅케이스 (로앤비, 일부 무료)
- **기업 법무팀**: BHSN All-in-One (유료)
