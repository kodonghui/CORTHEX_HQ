### 상세 변경 내역

**문제**: 3라운드 처장급 토론에서 처장들이 뻔한 말만 하고, 실질적으로 유용한 분석이 나오지 않았음

**원인 분석**:
1. 1라운드 프롬프트가 "독립적인 전문 의견을 제시하세요"로 너무 막연 → 교과서 내용만 나옴
2. 모든 처장에게 동일한 지시 → CTO가 기술분석, CSO가 사업분석을 안 하고 다 비슷한 말
3. 반박 라운드가 "문제점 1가지 지적"만 요구 → "일리 있지만"으로 시작하는 빈 양보만 나옴
4. 에이전트가 "보고서 모드"로 응답 → 토론이 아니라 형식적 보고서가 나옴

**해결**:
1. **처장별 맞춤 분석 관점(`_DEBATE_LENSES`)** 추가 — CIO는 재무/ROI, CTO는 기술 실현가능성, CSO는 시장/전략, CMO는 고객/CAC, CLO는 법적리스크, CPO는 제품/콘텐츠 관점
2. **1라운드 발언 규칙 강화** — 결론 먼저, 모호한 표현 금지, 구체적 수치/사례/기한 요구, 300~800자 제한
3. **"임원 토론 모드" 선언** — 보고서 틀(## 처장 의견 등) 사용 금지, 토론 발언으로 답하도록 지시
4. **반박 라운드 3단계 필수** — ① 다른 처장 이름 거론하여 구체 반박 ② 아무도 안 한 새 관점 추가 ③ 찬성/반대/조건부 최종 입장 표명
5. **비서실장 종합 구조화** — 한줄결론 → 핵심쟁점(표) → 전원합의 → CEO 결정사항(A/B) → 비서실장 권고
