## 뭘 했나? (변경 사항)

### 새로 만든 도구 파일 12개

| 번호 | 파일 | 도구 이름 | 뭐 하는 도구인가 | 사용하는 API |
|---|---|---|---|---|
| 1 | `src/tools/naver_news.py` (뉴스 검색 도구) | naver_news | 최신 뉴스 기사를 검색해줌 | 네이버 검색 API |
| 2 | `src/tools/ecos_macro.py` (거시경제 도구) | ecos_macro | 금리, GDP, 환율 등 경제 지표 조회 | 한국은행 ECOS API |
| 3 | `src/tools/kr_stock.py` (주식 데이터 도구) | kr_stock | 한국 주식 시세, 차트, 기술지표 계산 | pykrx (한국거래소 데이터) |
| 4 | `src/tools/dart_api.py` (재무제표 도구) | dart_api | 기업 재무제표, 공시, 회사정보 조회 | 금융감독원 DART API |
| 5 | `src/tools/naver_datalab.py` (트렌드 도구) | naver_datalab | 키워드 검색량 추이 분석 | 네이버 데이터랩 API |
| 6 | `src/tools/public_data.py` (공공데이터 도구) | public_data | 인구, 고용, 물가 등 정부 통계 조회 | 공공데이터포털 API |
| 7 | `src/tools/law_search.py` (법령 검색 도구) | law_search | 법령, 판례 검색 | 국가법령정보센터 API |
| 8 | `src/tools/kipris.py` (특허 검색 도구) | kipris | 특허, 상표, 디자인 검색 | KIPRIS Plus API (특허정보원) |
| 9 | `src/tools/github_tool.py` (깃헙 도구) | github_tool | GitHub 이슈, PR, 커밋 조회 | GitHub REST API |
| 10 | `src/tools/code_quality.py` (코드 품질 도구) | code_quality | 보안 검사, 코드 스타일, 테스트 실행 | bandit + ruff + pytest (로컬) |
| 11 | `src/tools/notion_api.py` (노션 도구) | notion_api | 노션 페이지 생성/수정/조회 | Notion API |
| 12 | `src/tools/doc_converter.py` (문서 변환 도구) | doc_converter | 마크다운 → PDF/Word/EPUB 변환 | pandoc (로컬) |

### 수정한 설정 파일 4개

| 파일 | 뭘 수정했나 |
|---|---|
| `config/tools.yaml` (도구 설정) | 12개 새 도구의 이름, 설명, 사용 모델 등록 |
| `src/tools/pool.py` (도구 등록소) | 12개 도구를 시스템에 연결 (import + 등록) |
| `config/agents.yaml` (에이전트 설정) | 18명의 에이전트에게 각자 역할에 맞는 도구를 배정하고, 사용법을 알려줌 |
| `pyproject.toml` (패키지 설정) | pykrx, pandas-ta, pandas, notion-client 라이브러리 추가 |
| `.env.example` (환경변수 예시) | DART_API_KEY, ECOS_API_KEY 등 8개 API 키 항목 추가 |
