### 🔴 심각 버그 3건 — 수정 완료

| # | 뭘 고쳤는지 | 이전 | 이후 |
|---|------------|------|------|
| 1 | **보내기 버튼 2개 겹침 제거** | 보내기 버튼이 2개 있고 HTML 태그(코드 구조)가 깨져 있었음 | 중복 버튼 삭제, 깔끔하게 1개만 |
| 2 | **SNS 연결 상태 API 추가** | SNS 탭에서 항상 "미연결"로 표시됨 | `/api/sns/oauth/status` API 새로 만들어서 7개 플랫폼 상태 확인 가능 |
| 3 | **작업 취소 실제 동작** | 취소 버튼 눌러도 화면만 바뀌고 실제로 안 멈춤 | 백엔드에서 실제로 작업 중단 처리 |

### 🟠 빠진 기능 7건 — 추가 완료

| # | 뭘 추가했는지 | 설명 |
|---|-------------|------|
| 4 | **출판·기록 본부(CPO) 조직도 추가** | 왼쪽 조직도에 초록색 "PB" 아이콘으로 출판 본부가 보임 |
| 5 | **도구 풀 50개+ 동적 표시** | 하드코딩(고정된) 5개 → 서버에서 실제 도구 목록을 불러와서 전부 표시 |
| 6 | **SNS 3개 플랫폼 추가** | 네이버 블로그, 네이버 카페, 다음 카페가 SNS 탭에 추가됨 (한글 이름으로 표시) |
| 7 | **CEO 만족도 통계 카드** | 대시보드(홈)에 만족도 %, 좋아요/아쉬워요 건수가 표시됨 |
| 8 | **예산 수정 UI** | 대시보드의 예산 카드에 "수정" 버튼 → 일일/월간 한도를 직접 변경 가능 |
| 9 | **작업 비교 기능 완성** | 작업 내역에서 2개 선택 → "비교하기" 누르면 나란히 비교하는 팝업(모달) 표시 |
| 10 | **예약 삭제** | 기존에 이미 동작 확인됨 (정상) |

### 🟡 UX 개선 8건 — 완료

| # | 뭘 개선했는지 | 설명 |
|---|-------------|------|
| 11 | **모바일 반응형** | 핸드폰 화면(768px 이하)에서 사이드바 자동 숨김, 카드가 1열로 쌓임 |
| 12 | **탭 정리 (5+더보기)** | 홈/명령/성능/작업내역/SNS만 기본 표시, 나머지는 "더보기" 드롭다운 메뉴 |
| 13 | **활동 로그 유지** | 새로고침(페이지 다시 열기)해도 최근 100건 로그가 남아 있음 (브라우저에 저장) |
| 14 | **작업 내역 20개씩** | 한 번에 전부 로드하지 않고 20개씩 나눠서 표시 + "더 보기" 버튼 |
| 15 | **대화 내보내기** | 명령 탭에서 "대화 내보내기" 버튼 → .md 파일(마크다운)로 다운로드 |
| 16 | **키보드 단축키** | Ctrl+K: 명령 입력창으로 바로 이동, Esc: 열린 팝업 닫기 |
| 17 | **완료 상태 5초 유지** | 에이전트 작업이 끝나면 초록불이 5초 동안 유지되어 "누가 일했는지" 볼 수 있음 |
| 18 | **라이트 모드 보완** | 라이트 모드에서 어색했던 투명도/색상 요소들 추가 수정 |
