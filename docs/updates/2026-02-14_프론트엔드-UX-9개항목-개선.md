## 변경 사항 요약

CEO 피드백 9개 항목을 모두 반영했습니다.

### 1. 모델 설정 정리 — 제조사별 그룹 분류
- **뭘 바꿨나**: 모델 선택 드롭다운(목록 상자)에서 "Anthropic (Claude)" / "OpenAI (GPT)" 로 나누어 보여줍니다
- **왜**: 11개 모델이 섞여있으면 뭐가 뭔지 알기 어려워서
- **추가**: 각 모델 옆에 등급 표시 (임원급, 매니저급, 전문가급, 실무급)

### 2. GPT Codex, Nano 삭제
- **뭘 바꿨나**: `models.yaml`에서 3개 모델 완전 삭제
  - `gpt-5-nano` (성능 낮음)
  - `gpt-5.1-codex` (코딩 전용, 에이전트에 불필요)
  - `gpt-5.2-codex` (코딩 전용, 에이전트에 불필요)
- **결과**: 8개 모델만 남음 (Claude 3개 + GPT 5개)

### 3. C-Level 사이드바 카드형 디자인
- **뭘 바꿨나**: CTO, CSO, CLO, CMO, CIO, CPO 각 부서의 사이드바 토글(접기/펼치기 버튼)을 카드형으로 완전히 리디자인
- **변경 내용**:
  - 부서별 고유 색상: CTO=시안, CSO=보라, CLO=초록, CMO=노랑, CIO=빨강, CPO=파랑
  - "AI 에이전트 N명" 표시 추가
  - 펼쳤을 때 왼쪽에 색상 라인 표시
  - 클릭하면 부서 색상으로 배경이 살짝 바뀜

### 4. 화면 대비 강화 (카드 배경 밝기)
- **뭘 바꿨나**: 카드 배경색과 테두리를 더 밝게 변경
  - 카드 배경(surface): `#1a2744` → `#1e2d4a` (밝기 약 15% 상승)
  - 카드 패널(panel): `#243352` → `#2a3d5e`
  - 테두리: `#475569` → `#4a6080`
  - 글래스 효과도 밝기 강화
- **왜**: 어두운 배경 위에 카드가 거의 안 보이는 문제 해결

### 5. 시스템 상태 카드 → API 키 상태
- **뭘 바꿨나**: 대시보드(전체 현황판)의 "시스템 정상/점검" 카드를 "API 키 연결 상태" 카드로 교체
- **표시 내용**: OpenAI 등록됨/미등록, Anthropic 등록됨/미등록
- **왜**: 기존 시스템 상태 카드는 용도가 불분명했음

### 6. 부서 이름 전체 변경 (~처 → ~팀)
- **변경 전 → 변경 후**:
  - 기술개발처 → **기술개발팀**
  - 사업기획처 → **전략기획팀**
  - 법무 IP처 → **법무팀**
  - 마케팅 고객처 → **마케팅팀**
  - 투자분석처 → **금융분석팀**
  - 출판·기록 → **콘텐츠팀**
- **수정 위치**: 사이드바 HTML, JavaScript, Python 백엔드, YAML 설정 파일 전부

### 7. CPO(콘텐츠팀) 품질검수 기준 추가
- **뭘 바꿨나**: 품질검수 설정에 콘텐츠팀(CPO) 전용 검수 기준 추가
- **검수 기준**: 정확성(35점), 가독성(30점), 완성도(35점)
- **수정 파일**: `quality_rules.yaml`, `quality_rules_manager.py`

### 8. 품질검수 편집 → 인라인 아코디언
- **뭘 바꿨나**: "편집" 버튼 누르면 해당 부서 바로 밑에 편집 영역이 펼쳐짐
- **이전**: 편집 버튼 누르면 화면 맨 아래에 편집기가 나타남 (스크롤 필요)
- **이후**: 부서별 편집/접기 토글 — 해당 항목 바로 아래에 나타남

### 9. 검수 모델 드롭다운 통일
- **뭘 바꿨나**: 검수 모델 선택도 에이전트 모델 선택과 동일하게 제조사별 그룹 분류 + 등급 표시 적용
