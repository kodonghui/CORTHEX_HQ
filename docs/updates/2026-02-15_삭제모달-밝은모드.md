# 삭제 확인 모달 밝은모드 테마 지원

## 버전
0.09.001

## 작업 날짜
2026-02-15

## 작업 브랜치
claude/fix-delete-modal-light-theme

## 변경 사항 요약

예약이나 워크플로우에서 "삭제" 버튼을 누르면 뜨는 확인 팝업(모달)이 **밝은 모드에서도 잘 보이도록** 수정했습니다.

### 문제
- 삭제 확인 팝업이 다크모드 색상(짙은 보라 배경, 연분홍 글자)으로 고정되어 있었음
- 밝은 모드에서는 어두운 배경이 그대로 나와서 디자인이 어울리지 않았음

### 해결
- Alpine.js의 `darkMode` 변수를 이용해서, 다크/라이트 모드에 따라 색상이 자동으로 바뀌도록 수정
- 다크모드: 기존 디자인 100% 유지 (어두운 보라 + 빨간 글로우 효과)
- 라이트모드: 흰색 배경 + 진한 빨간 아이콘 + 부드러운 그림자

### 수정한 파일
- `web/templates/index.html` (홈페이지 메인 화면) — 삭제 확인 모달 부분만 수정

### 변경된 요소 (다크 → 라이트)
| 요소 | 다크모드 | 라이트모드 |
|------|---------|-----------|
| 배경 오버레이 | 검정 60% | 검정 30% |
| 카드 배경 | 짙은 보라색 반투명 | 흰색 95% |
| 쓰레기통 아이콘 | 연분홍 + 빨간 빛 | 진한 빨간 + 은은한 그림자 |
| 삭제할 항목 이름 | 연분홍색 | 짙은 빨간색 |
| 안내 문구 | 연보라 회색 | 진한 회색 |
| 삭제하기 버튼 | 빨간 + 강한 빛 | 빨간 + 부드러운 그림자 |
| 취소 버튼 | 흰색 반투명 테두리 | 검정 반투명 테두리 |
