# UX/UI 5대 문제 수정

## 작업 날짜
2026-02-13

## 작업 브랜치

## 변경 사항 요약

### 뭘 했는지
CEO가 직접 화면을 써보면서 발견한 **5가지 문제**를 전부 수정했습니다.

---

### 문제 1: 색 대비가 없어서 거의 안보임

**원인**: 다크모드에서 배경색(#0a0e1a)과 테두리색(#1e2d4a)의 차이가 거의 없었고, 보조 텍스트도 흐려서 글자가 안 읽혔습니다. 패널(상자)끼리도 구분이 안 됐습니다.

**해결 방법**:

| 요소 | 수정 전 | 수정 후 | 효과 |
|------|---------|---------|------|
| 보조 텍스트 | #64748b (어두움) | #94a3b8 (밝게) | 글자가 잘 읽힘 |
| 테두리 | #1e2d4a (거의 안보임) | #334155 (밝게) | 경계선이 보임 |
| 패널 배경 | #111827 | #151d2e | 패널이 구분됨 |
| 사이드바 배경 | #1a2235 | #1e2940 | 사이드바가 구분됨 |
| 유리 효과 | 투명도 70% | 투명도 85% | 상자가 확실히 보임 |
| 격자 무늬 | 투명도 3% | 투명도 6% | 배경 격자가 보임 |
| 마우스 올렸을 때 | 투명도 2~3% | 투명도 5~6% | 호버 효과가 보임 |

---

### 문제 2: "사무실" 버튼 눌러도 화면이 안 바뀜

**원인**: 화면 전환 시스템에 버그가 있었습니다. "사무실"을 눌러도 탭 콘텐츠(홈/명령 등)가 사라지지 않아서 사무실 화면이 **뒤에 숨어서** 안 보였습니다.

**해결 방법**: 모든 탭 콘텐츠(10개)에 "채팅 모드일 때만 보이기" 조건을 추가했습니다. 사무실이나 대시보드를 선택하면 탭 콘텐츠가 사라지고 해당 화면이 표시됩니다.

---

### 문제 3: "대시보드" 버튼 눌러도 빈 화면

**원인**: 대시보드 HTML(화면 코드)이 깨져있었습니다. "로딩 중..." 글자만 있고 실제 내용이 없었고, 닫는 태그도 빠져서 화면 구조가 망가져 있었습니다.

**해결 방법**: 대시보드를 처음부터 다시 만들었습니다:
- **요약 카드 4개**: 오늘 작업 수, 완료, 실패, 진행 중
- **비용/에이전트 정보**: 오늘 총 비용, 에이전트 수, 시스템 상태
- **예산 진행 바**: 일일/월간 예산 사용률 (80% 초과 시 빨간색 경고)
- **품질 통계**: 합격률, 총 검수 건수, 불합격 건수
- **최근 완료 작업**: 최근 5건의 작업 결과
- **자동 새로고침**: 대시보드가 열려있으면 30초마다 데이터 갱신

---

### 문제 4: 에이전트 이름/역할/부서가 빠져있음

**원인**: 백엔드(서버)에서 이름, 역할, 부서, 모델명을 다 보내주는데, 화면에는 이름만 표시하고 나머지는 버리고 있었습니다.

**해결 방법**:
- **사이드바**: 에이전트 이름 아래에 역할 + 사용 모델을 작은 글씨로 표시
  - 예: `비서실장` → `비서실장` + `CEO 직속 비서 · gpt-5`
- **사무실 뷰**: 에이전트 카드에 역할 텍스트 추가
- **상태 요약 바**: 사이드바 상단에 "🟢 3 활동 · 🟡 2 대기 · ⚫ 20 비활성" 표시 추가

---

### 문제 5: 모델별 추론 정도가 다른데 반영 안 됨

**원인**: 추론 정도 설정 화면에 "낮음/보통/높음" 3개 버튼이 고정으로 박혀있었습니다. 하지만 실제로는:
- GPT-5-2: `없음/낮음/보통/높음/최고` 5단계 지원
- GPT-5-2-pro: `보통/높음/최고` 3단계만 지원
- Claude Haiku: 추론 지원 안 함

**해결 방법**:

| 수정 대상 | 내용 |
|-----------|------|
| `config/models.yaml` (모델 설정) | 각 모델의 실제 지원 추론 레벨로 업데이트 |
| `web/app.py` (서버) | 유효성 검사에 `none`, `minimal`, `xhigh` 추가 |
| `index.html` (화면) | 고정 3버튼 → 모델에 따라 동적 생성 |

**추론 레벨 표시**:

| 레벨 | 표시 | 설명 |
|------|------|------|
| none | ⛔ 없음 | 추론 비활성화 |
| minimal | 💨 최소 | 아주 빠름 |
| low | ⚡ 낮음 | 빠르고 저렴 |
| medium | 🧠 보통 | 균형잡힌 성능 |
| high | 🔬 높음 | 정확하지만 느림 |
| xhigh | 🔥 최고 | 최대 깊이, 매우 비쌈 |

추론을 지원하지 않는 모델은 "이 모델은 추론 정도 설정을 지원하지 않습니다" 메시지가 표시됩니다.
모델을 변경하면 추론 옵션도 자동으로 갱신됩니다.
"초기화" 버튼으로 기본값으로 되돌릴 수 있습니다.
