## 변경 사항 요약

CSO(사업기획처장) 부서에서 사용할 수 있는 전문 도구 6개를 새로 만들었습니다.
이 도구들은 경쟁사 분석, 앱 리뷰 수집, 유튜브 분석, 정부 지원금 검색, 네이버 플레이스 리뷰 수집, 학술 논문 검색 기능을 제공합니다.

### 새로 만든 파일

| 파일 | 설명 |
|------|------|
| `src/tools/competitor_monitor.py` | 경쟁사 웹사이트 변화 감지기 — 경쟁사 웹사이트를 등록하면, 페이지 내용이 바뀔 때 자동으로 알려주고 어떤 변경이 사업적으로 의미 있는지 분석해줌 |
| `src/tools/app_review_scraper.py` | 앱 리뷰 수집기 — 구글 플레이스토어에서 앱 리뷰를 대량으로 가져와서, 별점 분포/불만 사항/칭찬 포인트를 자동 분석해줌 |
| `src/tools/youtube_analyzer.py` | 유튜브 채널 분석기 — 유튜브 채널의 영상 데이터(조회수, 제목 등)를 수집해서 어떤 콘텐츠가 잘 되는지 전략 분석해줌 |
| `src/tools/subsidy_finder.py` | 정부 지원금 찾기 — 기업마당/K-Startup에서 우리 회사가 받을 수 있는 정부 지원사업을 자동 검색하고 맞춤 추천해줌 |
| `src/tools/naver_place_scraper.py` | 네이버 플레이스 리뷰 수집기 — 네이버 지도에서 매장/학원의 리뷰를 수집하고, 고객 만족/불만 요인을 분석해줌 |
| `src/tools/scholar_scraper.py` | 학술 논문 검색기 — Google Scholar에서 논문을 검색하고, 연도별 논문 발표 추이를 분석하여 사업에 활용할 학술 근거를 제공 |

### 수정한 파일

| 파일 | 설명 |
|------|------|
| `src/tools/pool.py` (도구 등록소) | 새로운 6개 도구를 시스템에 등록 — 이 파일에 등록해야 에이전트(AI 직원)들이 도구를 사용할 수 있음 |
| `config/tools.yaml` (도구 설정 파일) | 6개 도구의 이름, 설명, 사용할 AI 모델 등 설정 추가 |
| `config/agents.yaml` (에이전트 설정 파일) | 어떤 에이전트(AI 직원)가 어떤 도구를 쓸 수 있는지 권한(허용 도구) 설정 |

## 각 도구 상세 설명

### 1. 경쟁사 웹사이트 감지기 (`competitor_monitor`)
- **뭘 하는 도구인가**: 경쟁사 홈페이지를 등록해두면, 내용이 바뀔 때 자동으로 알려줌
- **어떻게 쓰는가**:
  - "메가로스쿨 홈페이지 감시해" → 등록 후 주기적으로 변경 확인
  - 가격 변경, 신규 서비스 추가 등을 자동 감지
- **기술적 동작**: 웹페이지 텍스트를 해시(고유번호) 값으로 저장 → 다음에 확인할 때 비교 → 다르면 변경으로 판단

### 2. 앱 리뷰 수집기 (`app_review_scraper`)
- **뭘 하는 도구인가**: 구글 플레이스토어에서 앱 리뷰를 대량 수집하고 분석
- **어떻게 쓰는가**:
  - "메가스터디 앱 리뷰 100개 가져와" → 리뷰 수집 + 별점 분포 + 불만 사항 정리
  - 두 앱의 리뷰를 비교 분석도 가능
- **필요한 것**: `google-play-scraper` 라이브러리 (없으면 설치 안내 메시지 표시)

### 3. 유튜브 채널 분석기 (`youtube_analyzer`)
- **뭘 하는 도구인가**: 유튜브 채널의 영상 데이터를 수집하고 콘텐츠 전략 분석
- **어떻게 쓰는가**:
  - "이 유튜브 채널 분석해" → 조회수 통계, 잘 되는/안 되는 영상 비교
  - 키워드 검색 결과 분석으로 콘텐츠 기회 발견
- **필요한 것**: `yt-dlp` 라이브러리 (없으면 설치 안내 메시지 표시)

### 4. 정부 지원금 찾기 (`subsidy_finder`)
- **뭘 하는 도구인가**: 기업마당, K-Startup에서 받을 수 있는 정부 지원사업 자동 검색
- **어떻게 쓰는가**:
  - "AI 교육 관련 정부 지원금 찾아" → 검색 + 추천
  - "우리 회사(창업 3년 이내, 교육업)에 맞는 지원금" → 맞춤 필터링

### 5. 네이버 플레이스 리뷰 수집기 (`naver_place_scraper`)
- **뭘 하는 도구인가**: 네이버 지도(플레이스)에서 매장/학원 리뷰 수집 + 분석
- **어떻게 쓰는가**:
  - "서울 LEET 학원 검색해" → 장소 목록
  - "이 학원 리뷰 분석해" → 만족/불만 요인 TOP 5

### 6. 학술 논문 검색기 (`scholar_scraper`)
- **뭘 하는 도구인가**: Google Scholar에서 논문 검색 + 트렌드 분석
- **어떻게 쓰는가**:
  - "법학적성시험 논문 검색해" → 관련 논문 목록
  - "이 분야 10년간 트렌드" → 연도별 논문 수 차트 + 분석
- **필요한 것**: `scholarly` 라이브러리 (없으면 직접 크롤링으로 대체)
