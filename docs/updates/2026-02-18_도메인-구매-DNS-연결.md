## 변경 사항 요약

### 도메인 구매
- **도메인**: `corthex-hq.com`
- **구매처**: Cloudflare Registrar (원가 판매, 마진 없음)
- **비용**: $10.46/년 (매년 동일 금액 자동 갱신)
- **DNS 연결**: `corthex-hq.com` → `158.179.165.97` (ARM 서버) 완료
- **접속 주소**: http://corthex-hq.com (기존 IP 주소와 동일하게 작동)

### 수정된 파일

| 파일 | 무슨 파일인가? | 뭘 바꿨나? |
|------|-------------|-----------|
| `CLAUDE.md` (작업 규칙 파일) | 클로드가 매번 읽는 규칙 파일 | 도메인 정보 추가, 구서버 IP 오류 수정 |
| `.env.example` (환경변수 예시 파일) | API 키 설정 예시 | Redirect URI 전부 `corthex-hq.com`으로 변경 |
| `docs/project-status.md` (프로젝트 현황 파일) | 클로드가 세션 시작 시 읽는 현황 파일 | 도메인 정보, 서버 IP 추가 |
