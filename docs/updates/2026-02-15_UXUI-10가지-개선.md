## 변경 사항 요약

CEO가 요청한 8가지 + 추가 2가지, 총 10가지 UX/UI를 한 번에 개선했습니다.

### 1. 다크모드 밝기 개선
- **이전**: 배경색 RGB(15,14,22) → 거의 검정, 너무 어두움
- **이후**: RGB(24,22,34) → "부드러운 밤하늘" 느낌
- surface, panel, border 색상도 함께 10~12 정도 밝게 조정
- 오로라 그라데이션 배경도 밝기 맞춤

### 2. 보내기 버튼 가시성 개선
- **이전**: 비활성 시 `gray-700` → 다크모드에서 안 보임, 라이트모드에서도 안 보임
- **이후**: 활성 = 파란→보라 그래디언트 + 발광 그림자 (shadow-lg), 비활성 = 파란색 반투명 테두리 + 연한 파란 배경
- 버튼 크기 w-11→w-12 (44px→48px)

### 3. 루브릭 편집란 글씨체
- 품질검수 루브릭 프롬프트 편집란에 JetBrains Mono 폰트 적용
- line-height 1.7, letter-spacing 0.01em으로 가독성 향상

### 4. 탭 전환 애니메이션
- 명령, 작업내역, 예약, 워크플로우, 지식, 아카이브, SNS 탭 전부에 `x-transition` 추가
- 진입: 아래에서 위로 살짝 올라오며 페이드인 (0.2초)

### 5. 아카이브 페이지 디자인 개선
- 제목에서 "(보고서 저장소)" 부제 제거
- 아이콘(서류함 SVG) + "총 N건" 카운터 추가
- 리스트형 → 2열 카드형 그리드로 변경
- 부서별 색상 구분 (기술=시안, 전략=보라, 법무=빨강, 마케팅=초록, 재무=노랑)
- 빈 화면: 떠다니는 아이콘 + 안내 문구

### 6. CEO 사무실 클릭 기능
- 사무실 탭의 CEO 카드: `cursor-default` → `cursor-pointer` + `@click="showCeoProfile = true"`
- 이제 사무실에서도 CEO 카드를 클릭하면 프로필 모달이 열림

### 7. 예산 대시보드 항상 표시
- **이전**: `x-show="budget && (budget.daily_limit || budget.monthly_limit)"` → 한도 미설정 시 안 보임
- **이후**: 항상 표시. 한도 미설정 시 "예산 한도가 설정되지 않았습니다" + "한도 설정하기" 버튼
- 대시보드에서도 "한도 수정" 버튼 추가

### 8. 빌드 상태 카드 정렬 통일
- 3개 카드 높이 통일 (min-h-[76px])
- 라벨을 상단, 값을 하단 중앙에 배치
- 각 카드에 아이콘 추가 (빌드=플라스크, 서버=서버, 시간=시계)
- 글자 크기 text-base로 통일

### 9. CORTHEX 로고 클릭 → 홈
- 좌측 상단 CORTHEX 로고 클릭 시 `viewMode='chat'; activeTab='command'`로 홈 이동
- 호버 시 opacity 살짝 줄어드는 효과 추가

### 10. 입력란 구분 강화 + 대화 내보내기 위치 이동
- **입력란**: `border-1` → `border-2` + `border-hq-border-light` + `shadow-sm`
- 입력바 영역 상단에 `border-t-2 border-hq-accent/20` + `shadow` 추가
- 라이트모드 입력란: 배경을 흰색(`rgb(255 255 255 / 0.9)`)으로 변경 + box-shadow 추가
- **대화 내보내기**: 입력란 위 별도 줄 → 프리셋 칩 영역 오른쪽 끝으로 이동
