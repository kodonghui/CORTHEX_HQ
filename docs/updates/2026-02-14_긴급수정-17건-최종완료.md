## 변경 사항 요약

### 수정한 파일
| 파일 | 설명 |
|------|------|

---

### 1. 화면 색상 전면 교체 (가장 긴급)
- **문제**: 화면이 너무 어둡고 글씨가 안 보였음
- **해결**: 배경색을 밝게 조정하고, 투명도가 낮았던 글씨 62곳을 전부 찾아서 진하게 변경

### 2. 사이드바 통일
- **문제**: 비서실/도구함에만 테두리(카드 스타일)가 없었음
- **해결**: 비서실과 도구함에 다른 부서와 같은 카드 스타일(테두리 + 아이콘) 적용

### 3. 매니저급 에이전트 추가
- **문제**: CTO, CSO, CLO, CMO, CIO가 사이드바에서 안 보였음
- **해결**: 각 부서 에이전트 목록에 매니저급 추가 (클릭하면 설정 가능)

### 4. CEO 프로필
- **문제**: CEO 카드를 눌러도 아무 반응 없었음
- **해결**: CEO 카드 클릭 시 프로필 모달(직속 보고 라인, 오늘 통계) 표시

### 5. 조직도 인원수 동적화
- **문제**: "25명"으로 하드코딩되어 있었음
- **해결**: 실제 에이전트 수를 자동 계산하여 표시

### 6. 에이전트 상관/부하 표시
- **문제**: 에이전트 기본정보에 상관/부하 관계가 안 보였음
- **해결**: 기본정보 탭에 상관(보고 대상)과 부하직원 목록 추가, 클릭하면 바로 해당 에이전트 설정으로 이동

### 7. 도구 클릭 기능
- **문제**: 도구함의 도구를 눌러도 아무 반응 없었음
- **해결**: 도구 클릭 시 상세 모달(이름, ID, 설명, 카테고리) 표시

### 8. 메뉴-콘텐츠 간격
- **문제**: 상단 메뉴 탭과 아래 콘텐츠가 너무 붙어 있었음
- **해결**: 탭과 콘텐츠 사이 여백 확대 (pt-2, pt-8)

### 9. 비서실 직원 역할 변경
- **문제**: 비서실 직원이 Worker(실무자)로 되어 있었음
- **해결**: Specialist(전문가)로 변경하고, CEO가 지정한 이름으로 개명
  - 보고요약 워커 → **브리핑 보좌관**
  - 일정관리 워커 → **작전 보좌관**
  - 연락담당 워커 → **부서 보좌관**

### 10. 에이전트 스킬 탭
- **문제**: 각 에이전트가 쓸 수 있는 도구를 알 수 없었음
- **해결**: 에이전트 설정 모달에 "스킬" 탭 추가 — 사용 가능 도구 목록 표시

### 11. 작업내역 통계 카드
- **문제**: 작업내역 탭에 검색바만 있었음
- **해결**: 상단에 통계 카드 4종 추가 (전체 작업 수, 완료, 실패, 총 비용)
- 빈 상태일 때 아이콘 + 안내문 + "명령하러 가기" 버튼 추가

### 12. SNS 연결 안내
- **문제**: SNS "연결하기" 버튼을 눌러도 아무 반응이 없었음
- **해결**:
  - 연결 상태 탭 상단에 "API 키가 필요합니다" 안내 메시지 추가
  - 연결 실패 시 친절한 에러 메시지 표시 (어디서 API 키를 설정해야 하는지 안내)

### 13. SNS 이벤트 설명
- **문제**: "이벤트" 탭이 뭔지 모르겠음
- **해결**: 이벤트 탭 상단에 설명문 추가 ("게시물 발행, 댓글 알림, 팔로워 변동 등이 기록됩니다")

### 14. OpenAI 추론 모델 추가
- **문제**: AI 모델이 Claude만 있었음 (OpenAI 없었음)
- **해결**: `models.yaml`에 3개 모델 추가
  - **o3**: $2/$8 (추론 특화, low/medium/high)
  - **o3-pro**: $20/$80 (최상위 추론, medium/high/xhigh)
  - **o4-mini**: $1.10/$4.40 (경량 추론, low/medium/high)
- 프론트엔드 모델 표시명과 추론 fallback에도 반영

### 15. 버전 업데이트
- **변경**: v0.4.0 → **v0.5.0**
