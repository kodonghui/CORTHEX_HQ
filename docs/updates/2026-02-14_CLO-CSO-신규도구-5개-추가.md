## 각 도구 상세 설명

### 1. 플랫폼 시장 조사기 (CSO 사업기획처)
- **뭘 하는 도구인가**: 크몽, 탈잉, 클래스101 같은 플랫폼에서 "LEET", "법학" 같은 키워드로 검색해서, 경쟁 서비스들의 가격/리뷰/판매량을 자동 수집합니다.
- **왜 필요한가**: 우리 서비스 가격을 정할 때, 경쟁자들이 얼마에 팔고 있는지 알아야 합니다.
- **기능 3가지**:
  - `search`: 플랫폼 검색 (크몽/탈잉/클래스101 선택 가능)
  - `analyze`: 수집 데이터 기반 시장 분석 (가격 포지셔닝, 경쟁 강도, 차별화 기회)
  - `price_range`: 가격대별 분포 분석 (5만원 이하, 5~10만원, 10~30만원, 30만원 이상)
- **사용 가능 에이전트**: CSO 처장, 시장조사 Specialist

### 2. 판례 트렌드 분석기 (CLO 법무IP처)
- **뭘 하는 도구인가**: "저작권 침해", "개인정보 유출" 같은 주제로 법원 판례를 대량 수집해서, 최근 몇 년간 판결이 어떤 방향으로 가고 있는지 분석합니다.
- **왜 필요한가**: 우리 사업과 관련된 소송 위험이 얼마나 되는지, 어떻게 대비해야 하는지 알 수 있습니다.
- **기능 3가지**:
  - `analyze`: 연도별 건수, 승소율, 자주 인용되는 법 조문, 키워드 분석
  - `summary`: 특정 판례 1건 상세 요약
  - `risk`: 특정 사업/행위의 법적 리스크 수준 평가 (상/중/하)
- **사용 가능 에이전트**: CLO 처장, 저작권 Specialist, 특허/약관 Specialist

### 3. 상표 유사도 검사기 (CLO 법무IP처)
- **뭘 하는 도구인가**: "CORTHEX" 같은 브랜드명을 입력하면, 이미 등록된 상표와 얼마나 비슷한지 자동 판별합니다.
- **왜 필요한가**: 상표 등록 전에 미리 유사 상표가 있는지 확인해야 등록 거절을 피할 수 있습니다.
- **유사도 판별 방법 3가지** (직접 구현, 외부 라이브러리 없음):
  1. 글자 유사도: 편집 거리(Levenshtein Distance) 알고리즘
  2. 발음 유사도: 한글 자모(초성/중성/종성) 분리 후 비교
  3. 외관 유사도: 영문 대소문자 + 한글↔영문 발음 변환 비교
- **종합 점수**: 글자 40% + 발음 40% + 외관 20% = 0~100점
- **위험도**: 80점 이상 위험, 60~80 주의, 40~60 보통, 40 미만 안전
- **사용 가능 에이전트**: CLO 처장, 저작권 Specialist, 특허/약관 Specialist

### 4. 계약서 자동 검토기 (CLO 법무IP처)
- **뭘 하는 도구인가**: 계약서 텍스트를 넣으면, 위험한 조항(예: 일방적 해지, 무제한 손해배상)을 자동으로 찾아냅니다.
- **왜 필요한가**: 계약서에 숨겨진 불리한 조항을 미리 발견해서 협상할 수 있습니다.
- **위험 패턴**: 총 26개 패턴 (높음 10개, 중간 8개, 참고 8개)
- **기능 3가지**:
  - `review`: 위험 조항 자동 탐지 + 필수 조항 누락 체크
  - `checklist`: 계약서 유형별 필수 조항 목록 (서비스이용약관, 업무위탁, 투자계약, 고용계약, NDA)
  - `compare`: 두 계약서 비교 분석
- **사용 가능 에이전트**: CLO 처장, 저작권 Specialist, 특허/약관 Specialist

### 5. 법령 변경 알리미 (CLO 법무IP처)
- **뭘 하는 도구인가**: "저작권법", "개인정보보호법" 같은 법률을 감시 목록에 등록하면, 법이 바뀔 때 자동으로 알려줍니다.
- **왜 필요한가**: 법이 바뀌면 우리 서비스를 수정해야 할 수도 있는데, 놓치면 법 위반이 될 수 있습니다.
- **기능 5가지**:
  - `watch`: 감시할 법령 등록
  - `unwatch`: 감시 해제
  - `check`: 등록된 법령의 변경 여부 확인
  - `list`: 현재 감시 중인 법령 목록
  - `recent`: 최근 개정된 주요 법령 목록 (교육/정보통신/금융 분야별)
- **사용 가능 에이전트**: CLO 처장, 저작권 Specialist, 특허/약관 Specialist
