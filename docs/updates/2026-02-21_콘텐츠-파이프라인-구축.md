### 📰 콘텐츠 파이프라인이란?
CEO가 버튼 하나 누르면 자동으로:
1. **분석** — AI가 최신 시장/AI 트렌드 분석 (Gemini 2.5 Flash)
2. **작성** — 분석 결과를 블로그 글로 변환
3. **편집** — 맞춤법, 가독성, SEO 자동 검수
4. **CEO 승인** — 대시보드에서 미리보기 후 승인/거절
5. **발행** — 네이버 블로그 + 인스타그램 + 뉴스레터 자동 발행

### 🔧 추가된 API 엔드포인트
| API | 설명 |
|-----|------|
| `GET /api/content-pipeline` | 파이프라인 현황 (토픽, 실행이력, 승인대기) |
| `POST /api/content-pipeline/run` | 파이프라인 실행 (topic 선택) |
| `POST /api/content-pipeline/{id}/approve` | CEO 승인 → 자동 발행 |
| `POST /api/content-pipeline/{id}/reject` | CEO 거절 |
| `GET /api/content-pipeline/{id}` | 상세 조회 |
| `POST /api/content-pipeline/config` | 설정 변경 |
