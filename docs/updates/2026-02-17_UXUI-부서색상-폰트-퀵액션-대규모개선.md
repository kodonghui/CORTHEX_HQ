## 변경 사항 요약

### 1. 작전일지 필터 드롭다운 → 커스텀 드롭다운
- **변경 전**: 브라우저 기본 `<select>` 드롭다운 (화살표가 너무 오른쪽에 붙어서 어색)
- **변경 후**: Alpine.js로 만든 커스텀 드롭다운 (둥근 모서리 + 애니메이션 + 체크표시)
- 파일: `web/templates/index.html` (작전일지 필터 섹션)

### 2. 로고 간격 조정
- **변경 전**: CORTHEX와 HQ 사이 글자 간격이 너무 넓었음 (`tracking-widest`)
- **변경 후**: 글자 간격 줄임 (`tracking-wider`)
- 파일: `web/templates/index.html` (좌측 상단 로고)

### 3. 부서 이름 통일 (좌측 사이드바 + 사무실 탭)
- **변경 전**: 좌측은 "~팀", 사무실은 "~처"로 이름이 달랐음
- **변경 후**: 전부 "~처"로 통일 (config/agents.yaml 기준)
  - 기술개발팀 → 기술개발처
  - 전략기획팀 → 사업기획처
  - 법무팀 → 법무·IP처
  - 마케팅팀 → 마케팅·고객처
  - 금융분석팀 → 금융분석처
  - 콘텐츠팀 → 출판기록처

### 4. 부서 색상 7색 시스템 적용
- **변경 전**: 여러 부서가 같은 색(예: 비서실=마케팅=노란색, LEETMaster=기술개발=시안색)
- **변경 후**: 7개 부서가 전부 고유한 색상을 가짐

  | 기술개발처 | hq-cyan | cyan-500 (청록색) |
  | 사업기획처 | hq-purple | indigo-500 (남색) |
  | 법무·IP처 | hq-green | emerald-500 (에메랄드) |
  | 마케팅·고객처 | hq-yellow | pink-500 (분홍색) |
  | 금융분석처 | hq-red | rose-600 (장미색) |

### 5. 사무실 탭 배경 통일

### 6. 출판기록처(CPO) 사무실 섹션 신규 추가

- **변경 후**: PUBLISHING OPS 섹션 추가 (teal-500 색상)

### 7. 밝은 모드 다크 라벤더
- **변경 전**: 밝은 모드가 너무 밝아서 눈이 아팠음
- **변경 후**: 배경을 어두운 라벤더색(보라 기운의 회색)으로 변경

### 8. 한글 폰트 전수검사 (font-mono 45곳 제거)
- **문제**: JetBrains Mono 폰트가 한글에 적용되면 글씨가 이상하게 보임
- **해결**: `font-mono` 클래스가 한글 텍스트에 적용된 곳 45곳을 찾아서 전부 제거
  - "빌드 #", "CEO 관제", "연결됨/끊김", "정상", "개", "초", "명", "건" 등
  - 대시보드 "실시간 운영 현황", "에이전트 대기 중" 등
  - Trading 섹션 한글 제목/라벨 전부 (총 자산, 보유 현금, 매매 전략 등)
  - 영어/숫자만 표시하는 곳은 font-mono 유지 (CORTHEX HQ, CEO, 빌드 번호, 달러 금액 등)

### 9. 퀵 액션 개선 (일일 루틴 + 최근 사용)
- **변경 전**: "전체 보고 / 배치 실행 / 도구 점검" 3개 버튼만 있었음
- **변경 후**:
  - **루틴/시스템 탭 토글**: 루틴 탭(🏢 전 부서 현황, 📅 오늘 일정, 💰 예산 현황) / 시스템 탭(🔧 도구 점검, ⚡ 배치 실행, 📋 배치 상태)
  - **최근 사용 명령**: 사용자가 보낸 명령이 자동으로 기록되어 최대 5개까지 표시 (localStorage에 저장, 새로고침해도 유지)
  - 대시보드 퀵 액션 카드 + 채팅 대기 화면 양쪽 모두 개선
