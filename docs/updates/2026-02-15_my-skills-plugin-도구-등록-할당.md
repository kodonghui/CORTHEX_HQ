# my-skills-plugin 88개 스킬 도구 등록 + 29개 에이전트 할당

## 작업 날짜
2026-02-15

## 작업 브랜치
claude/add-skills-plugin-tools

## 변경 사항 요약

### 뭘 했는지
my-skills-collection 플러그인에 포함된 **88개 전문 스킬**을 CORTHEX HQ의 도구 시스템에 등록하고, **29명의 에이전트** 각각에게 역할에 맞는 스킬을 할당했습니다.

### 수정한 파일
1. `config/tools.yaml` (도구 설정 파일) — 88개 스킬 도구 추가
2. `config/agents.yaml` (에이전트 설정 파일) — 29개 에이전트에 스킬 도구 할당
