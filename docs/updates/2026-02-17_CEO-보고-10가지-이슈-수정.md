### 이슈별 수정 내역

| # | 문제 | 원인 | 해결 |
|---|------|------|------|
| 1 | `/` 명령어 팝업 한국어 글씨가 안 예쁨 | 폰트 크기가 너무 작음 (11px) | 13px로 키우고 팝업 너비도 넓힘 |
| 2 | 작전일지 탭 클릭해도 안 켜짐 | 한번 로딩 실패하면 다시 시도 안 함 | 탭 클릭할 때마다 무조건 다시 로딩 + 에러 표시 + "다시 시도" 버튼 추가 |
| 3 | 작전일지 디자인이 별로 | 카드가 크고 정보 밀도가 낮음 | 세로 타임라인 뷰로 전면 교체 — 날짜별 그룹(오늘/어제/날짜) + 타임라인 점 + 한 줄 정보 |
| 4 | 섹션 제목 글자 사이 간격이 넓음 | `tracking-wider` (자간 0.05em) | `tracking-wide` (자간 0.025em)로 절반 줄임 |
| 5 | CIO 시그널에 삭제 기능 없음 | 삭제 버튼과 API가 없었음 | × 삭제 버튼 추가 + `DELETE /api/trading/signals/{id}` API 추가 |
| 6 | 시세 조회 실패 메시지가 불친절 | "시세 조회 실패"만 표시 | 구체적 원인 표시 (예: "yfinance 미설치", "네트워크 오류") |
| 7 | 관심종목 차트 아이콘이 안 보임 | 📈 이모티콘이 작고 50% 투명 | SVG 아이콘으로 교체 + 투명도 70%로 올림 |
| 8 | 밝은 모드가 너무 밝음 | 배경색 rgb(232,228,240) | 15% 어둡게 → rgb(218,214,228) |
| 9 | CIO 매매시그널 탭에도 삭제 필요 | #5와 같은 문제 | #5와 함께 해결 — 두 곳 모두 삭제 버튼 추가 |
| 10 | 자동화 탭의 `{prev_result}` 안내가 불친절 | 텍스트 설명만 있었음 | "이전 결과 삽입" 버튼 추가 + 단계 간 화살표 표시 |
