## 작업 날짜
2026-02-14

## 작업 브랜치
claude/corthex-cmo-tools-wxvbb

## 변경 사항 요약

CMO(마케팅·고객처)팀이 사용할 수 있는 전문 도구 5개를 새로 만들었습니다.
이 도구들은 마케팅 전략 수립, 고객 여론 파악, SNS 운영 등에 활용됩니다.

### 새로 만든 파일

| 파일 | 설명 |
|------|------|
| `src/tools/seo_analyzer.py` | SEO 분석기 — 웹사이트가 구글/네이버 검색에 잘 노출되는지 14개 항목으로 점검하고 100점 만점으로 점수를 매겨주는 도구 |
| `src/tools/sentiment_analyzer.py` | SNS 감정 분석기 — 특정 키워드(예: "CORTHEX")에 대해 네이버 뉴스/블로그에서 사람들이 긍정적으로 말하는지, 부정적으로 말하는지 자동으로 분석하는 도구 |
| `src/tools/hashtag_recommender.py` | 해시태그 추천기 — 인스타그램/유튜브/틱톡 게시물에 달 해시태그를 자동으로 추천해주는 도구. 대형/중형/소형 해시태그를 최적 비율로 조합 |
| `src/tools/email_optimizer.py` | 이메일 마케팅 최적화기 — 마케팅 이메일 제목의 효과를 100점 만점으로 예측하고, 더 좋은 제목 대안을 만들어주는 도구 |
| `src/tools/competitor_sns_monitor.py` | 경쟁사 SNS 모니터 — 경쟁사의 블로그/인스타/유튜브 활동을 자동으로 수집하고 전략을 분석하는 도구 |

### 수정한 파일

| 파일 | 설명 |
|------|------|
| `src/tools/pool.py` | 도구 등록 관리자 — 새로 만든 5개 도구를 시스템에 등록 |
| `config/tools.yaml` | 도구 설정 파일 — 5개 도구의 이름, 설명, AI 모델 설정 추가 |
| `config/agents.yaml` | 에이전트(직원) 설정 파일 — CMO팀 에이전트 4명에게 새 도구 사용 권한 부여 |

### 각 도구의 주요 기능

#### 1. SEO 분석기 (`seo_analyzer`)
- `audit`: 웹사이트 SEO 종합 점검 (14개 항목, 100점 만점)
- `keywords`: 특정 키워드가 페이지에 얼마나 나오는지 밀도 분석
- `compare`: 두 웹사이트의 SEO 점수를 비교

#### 2. SNS 감정 분석기 (`sentiment_analyzer`)
- `analyze`: 키워드에 대한 긍정/부정/중립 여론 분석
- `trend`: 시간별 여론 변화 추이
- `report`: 종합 여론 보고서 (뉴스 + 블로그 통합)

#### 3. 해시태그 추천기 (`hashtag_recommender`)
- `recommend`: 주제와 플랫폼에 맞는 해시태그 세트 자동 생성
- `analyze`: 특정 해시태그의 인기도/경쟁도 분석
- `trending`: 카테고리별 인기 해시태그 목록

#### 4. 이메일 마케팅 최적화기 (`email_optimizer`)
- `analyze`: 이메일 제목 효과 점수 산정 (8개 규칙)
- `suggest`: 기존 제목의 개선안 생성
- `ab_test`: A/B 테스트용 제목 쌍 자동 생성

#### 5. 경쟁사 SNS 모니터 (`competitor_sns_monitor`)
- `add`/`remove`: 감시 대상 경쟁사 추가/제거
- `check`: 등록된 경쟁사의 최근 SNS 활동 확인
- `report`: 경쟁사 SNS 전략 종합 보고서
- `list`: 감시 중인 경쟁사 목록
