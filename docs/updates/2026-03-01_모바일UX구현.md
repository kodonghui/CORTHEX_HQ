## 변경 파일
- `web/static/css/corthex-styles.css` — 모바일 CSS 추가

## 구현 내용

### Bottom Nav 5탭 교체
- 기존: 작전현황/사령관실/전략실/통신로그/더보기
- 변경: **사령관실/전략실/작전일지/기밀문서/통신국**
- 통신국 탭에 pending 배지 (빨간 숫자) 추가

### 미처리 배너
- 통신국(SNS)에 승인 대기 항목이 있을 때 채팅 영역 상단에 노란 배너 표시
- 탭하면 통신국 탭으로 즉시 이동

### 딥링크
- `?tab=sns` 등 URL 파라미터로 해당 탭 자동 이동
- 300ms 딜레이 (Alpine.js 초기화 대기)

### CSS 추가
- `.mobile-pending-banner` — 노란 경고 배너
- `.mobile-nav-badge` — Bottom Nav 빨간 배지
- `.typing-dots` — 타이핑 인디케이터 애니메이션
- `.sns-card-mobile`, `.sns-approve-btn` — 통신국 카드 최적화
