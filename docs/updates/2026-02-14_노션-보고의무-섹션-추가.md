# 에이전트 29개 Soul 파일에 노션 보고 의무 섹션 추가

## 작업 날짜
2026-02-14

## 작업 브랜치
claude/notion-report-storage-BZPaw

## 변경 사항 요약

### 뭘 했는지
29개 에이전트 soul 파일(에이전트의 역할/규칙을 정의하는 설정 파일) 각각의 맨 끝에
**"📤 노션 보고 의무"** 섹션을 추가했습니다.

이 섹션이 하는 일: 각 에이전트가 작업을 끝내면 **반드시 노션 데이터베이스에 보고서를 제출**하도록 규칙을 추가한 것입니다.

### 왜 했는지
에이전트들이 작업을 하고 나서 보고서를 노션에 남기지 않으면,
CEO가 "얘네가 뭘 했는지" 확인할 방법이 없었습니다.
이제 모든 에이전트가 작업 후 자동으로 노션 DB에 보고서를 올리게 됩니다.
