# 리트마스터 3차 검수 루브릭 (원본)

> CEO 제공 (2026-02-23). CORTHEX 투자분석 블라인드 평가 루브릭의 원본 기반.

## 평가 원칙
1. 배경지식(VI. 배경지식 소개)은 평가 대상에서 제외한다. 단, 배경지식을 본문 판정 근거로 사용하면 Q2에서 감점한다.
2. Q1(정확성)이 1점이면 Q0(전체 만족도)는 최대 2점으로 제한한다.

## 평가 차원 (10개)

### 【Q0. 전체 만족도】 가중치: 자동계산 또는 별도평가
- 1점: 배포 불가 (정답 오류, 근거 부족, 형식 붕괴)
- 3점: 배포 가능하나 편집 필요 (일부 비약, 누락)
- 5점: 무편집 배포 가능 (정확, 근거기반, 가독성 우수)

### 【Q1. 정확성】 가중치: 20%
- 1점: 정답 또는 핵심 판정(참/거짓)이 틀림
- 3점: 결론은 맞으나 중간 적용에 비약/부분 오류
- 5점: 모든 판정이 지문 요건 적용과 일치

### 【Q2. 지문 근거성】 가중치: 15%
- 1점: 지문 밖 상식/이론을 판정 근거로 사용 ("일반적으로...", "통념상...")
- 3점: 주된 근거는 지문이나 일부가 상식에 기댐
- 5점: 모든 핵심 판단이 지문 문구/요건에 직접 매핑됨

### 【Q3. 언어 정밀성】 가중치: 15%
- 1점: 주어 바뀜(군인↔의무복무자), 범위 확장/축소, "할 수 있다"를 "된다"로 오독
- 3점: 큰 실수는 없으나 한두 번 흔들림
- 5점: 주어/시점/양태(간주/재량/예외) 체크를 체계적으로 수행

### 【Q4. 논리 구조】 가중치: 10%
- 1점: 왜 그렇게 판정되는지 재현 불가
- 3점: 흐름은 있으나 중간 연결이 자주 생략
- 5점: 요건 추출 → 대입 → 결론이 단계별로 선명

### 【Q5. 완결성】 가중치: 10%
- 1점: 필요한 보기/선지 누락, 템플릿 핵심 항목 다수 누락
- 3점: 다 다루나 일부가 매우 피상적
- 5점: 모든 보기/선지를 일관된 깊이로 처리, 결정적 포인트 제시

### 【Q6. 오답 배제 논리】 가중치: 15%
- 1점: 오답이 왜 오답인지가 "다름/그럴듯" 수준
- 3점: 오답 근거는 있으나 즉시 컷이 안 되거나 장황
- 5점: 오답 각각을 한 줄 포인트로 자름 (예: "ㄴ은 '군인'이라 직업군인 반례로 즉시 컷")

### 【Q7. 교육적 효과】 가중치: 5%
- 1점: 정답만 말하고 재사용 규칙 없음
- 3점: 팁이 있으나 일반화가 약함
- 5점: 함정 패턴 + 오답노트(흔들린 지점/원인/배제 근거)가 구체적이고 재사용 가능

### 【Q8. 가독성/간결성】 가중치: 5%
- 1점: 문장 과밀/중복/지시어 난무로 읽는 비용이 큼
- 3점: 읽히지만 일부 장황/반복
- 5점: 정보 밀도 높고 반복 최소, 한 번 읽어 이해 가능

### 【Q9. 형식 준수】 가중치: 5%
- 1점: 단일셀 규칙 위반(실제 줄바꿈/탭), 금지된 표/마크다운
- 3점: 큰 틀은 맞으나 일부 누락/사소한 위반
- 5점: 템플릿 구조, 출력 제약 정확히 준수

## 총점 계산
```
총점 = (Q1×20 + Q2×15 + Q3×15 + Q4×10 + Q5×10 + Q6×15 + Q7×5 + Q8×5 + Q9×5) / 100
```
