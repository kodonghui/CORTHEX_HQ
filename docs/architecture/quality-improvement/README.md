## 핵심 설계 원칙 (CEO 아이디어 기반)

1. **처장 = 5번째 분석가**: 취합만 하지 않고 독자 분석도 병렬 수행
2. **부서 간 협업 예절**: 다른 부서 결과물은 감사히 받아 재가공. QA 면제
3. **빨간펜 학습**: 반려 교훈을 메모장(warnings)에 저장 → 자동 주입
4. **Soul 진화**: 매주 반려 패턴 분석 → soul 개선 제안 → 대표님 승인
5. **스폰 라우팅**: 같은 팀=직접, 다른 팀=팀장 경유, 동면=차단
