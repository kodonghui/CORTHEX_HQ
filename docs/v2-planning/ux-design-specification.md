# UX Design Specification — CORTHEX v2

**Author:** Elddl
**Date:** 2026-03-03

---

## Design Direction

| 항목 | 결정 |
|------|------|
| **스타일** | 모던 SaaS (Linear/Notion 스타일) |
| **느낌** | 깔끔 + 여백 + 집중. 정보 과밀 금지 |
| **테마** | 다크/라이트 둘 다 (처음부터) |
| **컴포넌트** | shadcn/ui (Tailwind CSS 기반, 코드 소유) |
| **레이아웃** | 좌측 사이드바 + 메인 콘텐츠 |

## 앱 구조

### 1. 유저 앱 (corthex-hq.com)

**모든 유저 동일 UI. 데이터/CLI만 다름.**

#### 레이아웃

```
┌──────────────────────────────────────────────┐
│  CORTHEX                        🌙 ⚙️ 👤    │  ← 헤더 (로고 + 테마 + 설정 + 프로필)
├──────────┬───────────────────────────────────┤
│ 사이드바  │  메인 콘텐츠                      │
│          │                                   │
│ 🏠 홈    │  안녕하세요 민지님 👋              │
│          │                                   │
│ 💬 채팅   │  ┌─────────────────────────────┐ │
│  ├ 비서장 │  │                             │ │
│  ├ A-1   │  │  [현재 선택된 탭/채팅 콘텐츠] │ │
│  └ A-2   │  │                             │ │
│          │  │                             │ │
│ 📄 보고서 │  │                             │ │
│ 📊 전략실 │  │                             │ │
│ 🔔 알림   │  └─────────────────────────────┘ │
│ ⚙️ 설정  │                                   │
├──────────┴───────────────────────────────────┤
│  상태바: 에이전트 3명 온라인 | CLI 연결됨     │
└──────────────────────────────────────────────┘
```

#### 화면 목록 (Phase별)

**P1:**

| 화면 | 설명 |
|------|------|
| **로그인** | ID/PW 입력. 풀스크린. 깔끔한 중앙 폼 |
| **홈** | "안녕하세요 {이름}님" + 내 팀 목록 + (P2부터) 완료 작업 알림 |
| **채팅** | 사이드바에서 에이전트 선택 → 메인에 채팅 UI. 멀티턴 + 세션 연속성 |
| **설정** | 개인 API key 등록 (KIS/노션/이메일/텔레그램) |

**P2:**

| 화면 | 설명 |
|------|------|
| **보고서** | 문서함. 내 보고서 목록 + "CEO에게 보고" 버튼 + 코멘트 |
| **전략실** | KIS 매매 현황. 유저별 개인 API 연동 |

**P3:**

| 화면 | 설명 |
|------|------|
| **대시보드** | 비용 모니터링 + 에이전트 상태 |
| **SNS** | 콘텐츠 생성 + 승인/반려 |
| **작전일지** | 활동 로그 |
| **사내 메신저** | 같은 회사 인간끼리 채팅 (채팅 탭에 통합) |

### 2. 관리자 콘솔 (admin.corthex-hq.com)

**별도 앱. CRUD 위주. 테이블/폼 중심.**

```
┌──────────────────────────────────────────────┐
│  CORTHEX ADMIN                    👤 로그아웃 │
├──────────┬───────────────────────────────────┤
│ 사이드바  │  메인 콘텐츠                      │
│          │                                   │
│ 🏢 회사   │  ┌─────────────────────────────┐ │
│ 👥 직원   │  │                             │ │
│ 🤖 에이전트│  │  [테이블 or 폼]             │ │
│ 🏛️ 부서  │  │                             │ │
│ 🔧 도구   │  └─────────────────────────────┘ │
│ 🔑 CLI   │                                   │
│ 📊 보고라인│                                   │
└──────────┴───────────────────────────────────┘
```

#### 관리자 화면 목록

| 화면 | 설명 |
|------|------|
| **회사 관리** | 회사 CRUD. 테이블 + 생성/편집 폼 |
| **직원 관리** | 유저 CRUD. ID/PW 발급. 부서 배정 |
| **에이전트 관리** | 에이전트 CRUD. 소울 편집기(마크다운) + 도구 체크박스 + CLI 연결 |
| **부서 관리** | 부서 CRUD. 소속 에이전트 목록 |
| **도구 관리** | 도구 목록. 본부별 ON/OFF 체크박스 |
| **CLI 관리** | CLI 토큰 등록/수정/삭제. 회사 공용 API key |
| **보고 라인** | H → 상위자 매핑. 드롭다운 선택 |

## 핵심 UX 패턴

### 채팅 UI (핵심 화면)

```
┌─────────────────────────────────────┐
│ 💬 A-1 (에이전트A-1)     온라인 🟢  │
├─────────────────────────────────────┤
│                                     │
│  [A-1] 안녕하세요 민지님, 어제      │
│  요청하신 삼성전자 보고서           │
│  완성했습니다. 확인하시겠어요?      │
│                       오전 9:02    │
│                                     │
│            [민지] 어, 고마워.       │
│            결론 부분 좀 더 보강해줘  │
│                       오전 9:03    │
│                                     │
│  [A-1] 네, 바로 수정하겠습니다 ⏳  │
│                       오전 9:03    │
│                                     │
├─────────────────────────────────────┤
│ 메시지 입력...              📎 ➤   │
└─────────────────────────────────────┘
```

- 에이전트 응답 스트리밍 (타이핑 애니메이션)
- 세션 연속성: 어제 대화가 그대로 보임
- 에이전트 상태: 온라인 🟢 / 작업중 🟡 / 오류 🔴

## 인터랙션 규칙

| 패턴 | 규칙 |
|------|------|
| **로딩** | 스켈레톤 UI (빈 박스 깜빡임). 스피너 금지 |
| **에러** | 인라인 에러 메시지 + 에러 코드 표시 |
| **빈 상태** | "아직 없습니다" + 행동 유도 버튼 |
| **실시간** | WebSocket으로 에이전트 상태/알림 자동 업데이트 |
| **반응형** | 데스크톱 우선. 모바일은 P5 |
| **접근성** | 키보드 네비게이션 + aria 라벨 기본 |
