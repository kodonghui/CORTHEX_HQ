## 1. 택시 비유 — 가장 중요한 핵심

```
오퍼스로 시작 = 고급 택시 타자마자 기본요금 15,000원
소네트로 시작 = 일반 택시 타자마자 기본요금 4,800원

같은 곳 가는데 고급 택시 탈 필요 있나?
→ 일반 택시 타고 가다가 "여기서부터는 고급 택시 필요합니다" 하면 그때 바꿔 타기
```

**현재 대표님 방식**:
매번 오퍼스(비싼 택시)로 시작 → 단순한 일도 다 오퍼스가 처리 → 토큰 빠르게 소진

**누나 방식**:
소네트(보통 택시)로 시작 → 소네트가 판단:
- "이건 제가 합니다" (소네트로 처리)
- "이건 오퍼스 불러야 합니다" (오퍼스로 에스컬레이션)
- "이건 하이쿠(알바)한테 시킵니다" (서브에이전트)

## 4. 플랜모드 — 자동 진입 트리거

누나: "문장에 설계/플랜/계획/구조 같은 키워드 포함되면 플랜모드로 자동 진입해야 함. 안 그러면 오퍼스가 멘땅에 헤딩하면서 토큰만 씀"

```
대표님이 "이거 만들어" → 클로드가 바로 코딩 시작 (❌ 토큰 낭비)
대표님이 "이거 설계해" → 클로드가 플랜모드 진입 → 계획 → 승인 → 실행 (✅)

CLAUDE.md에 규칙: "설계/플랜/계획/구조/아키텍처" 포함 시 자동 플랜모드
```
