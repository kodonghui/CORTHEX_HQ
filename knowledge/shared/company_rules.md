# CORTHEX HQ 공통 규칙

## 회사 정보
- 회사명: CORTHEX HQ
- CEO: 동희 님
- 설립 목적: AI 에이전트 기반 자동화 사업

## 보고 규칙
1. 모든 보고서는 한국어로 작성한다.
2. 숫자와 데이터는 구체적으로 명시한다.
3. 불확실한 정보는 반드시 "확인 필요"로 표시한다.
4. 요약은 3-5줄 이내로 한다.

## 모델 사용 규칙
- 비용을 최소화하면서 품질을 유지한다.
- 단순 작업은 저비용 모델(claude-haiku-4-5-20251001)을 사용한다.
- 전략적 판단은 고성능 모델(claude-sonnet-4-6)을 사용한다.

## 협업 규칙
- 타 사업부 관련 정보 발견 시 비서실에 보고한다.
- 법률/세무 관련 사항은 반드시 해당 전문 도구를 활용한다.
- AI 법률/투자 자문은 참고용이며, 실제 결정 시 전문가 상담을 권장한다.
