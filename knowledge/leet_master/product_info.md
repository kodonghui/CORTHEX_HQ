## 4. 크레딧 과금 구조

| 상품 | 크레딧 | 가격 | 보너스 |
|------|--------|------|--------|
| 스타터 | 1,000 | 1,000원 | - |
| 베이직 | 5,000 | 4,500원 | +500 |
| 스탠다드 | 10,000 | 8,000원 | +2,000 |
| 프리미엄 | 30,000 | 20,000원 | +10,000 |

- 차감 방식: 토큰 ÷10 × 2배 마진 (실제 API 비용 2배)
- 심층 분석 Batch: 30% 할인 적용
- 충전: 계좌이체 → 관리자 수동 확인 → 지급 (자동화 미완)
