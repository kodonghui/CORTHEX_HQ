## 2. CORTHEX HQ란?

**CORTHEX HQ**는 "AI 에이전트 회사"입니다.

일반적인 AI 챗봇은 하나의 AI에게 모든 것을 물어봅니다.
CORTHEX HQ는 다릅니다. **29명의 AI 에이전트를 실제 회사 조직도처럼 배치**하고,
대표(사용자)가 한국어로 명령하면, 비서실장이 판단해서 적절한 부서에 자동 배분하고,
전문가들이 각자 맡은 분야에서 병렬로 작업한 뒤, 결과를 종합 보고서로 합쳐서 돌려줍니다.

### 핵심 콘셉트: "AI 직원에게 일을 시키는 것"

```
일반 AI 챗봇:    사용자 <-> AI 1명 (만능이지만 깊이 없음)

CORTHEX HQ:     대표 (사용자)
                  |-> 비서실장 (명령 분석 + 배분)
                       |-> CTO (기술팀 5명에게 병렬 지시)
                       |-> CSO (기획팀 3명에게 병렬 지시)
                       |-> CLO (법무팀 2명에게 병렬 지시)
                       |-> CMO (마케팅팀 3명에게 병렬 지시)
                       |-> CIO (투자분석팀 5명에게 병렬 지시)
                       |-> CPO (출판기록팀 3명에게 병렬 지시)
```

## 3. 왜 만들었나?

### 문제를 느낀 순간

1. **하나의 AI에게 복잡한 일을 시키면 깊이가 없다**
   - "인스타 마케팅 전략 만들어줘"라고 하면 피상적인 답변만 나옴
   - 실제 회사에서는 리서치팀, 콘텐츠팀, 커뮤니티팀이 각각 깊이 작업한 뒤 합치는데, AI 1명에게는 그게 불가능

2. **전문 분야가 다른 업무를 하나의 AI가 다 커버할 수 없다**
   - 주식 분석, 마케팅, 법무, 기술 개발은 완전히 다른 도메인
   - 하나의 시스템 프롬프트로 모든 전문성을 주입하면 품질이 떨어짐

3. **"내가 사장이고, AI가 직원이면 어떨까?"**
   - 명령 -> 배분 -> 실행 -> 종합 -> 보고의 체계적 흐름
   - 누가 무슨 작업을 했는지 추적 가능
   - 비용이 얼마나 들었는지 실시간 모니터링

### Day 5 (2026-02-14): 서버 배포 + 자동화 — 가장 바빴던 날

이 날은 무려 **32개의 작업 기록**이 생겼습니다. 하루에 32번의 개선!

핵심 성과:
- **Oracle Cloud 24시간 무료 서버 구축**: 회사 컴퓨터가 꺼져도 웹사이트 접속 가능

- **Cloudflare Tunnel 외부 접속**: 집에서도 회사 서버 접속 가능
- **다크/라이트 테마 전면 수정**: 다크모드에서 글자가 안 보이는 문제 해결
- **부서별 전문가 도구 30개+ 추가**: CTO, CSO, CLO, CMO, CIO, CPO 각 부서에 5~7개씩

**이 날의 사고:**
- 서버에 코드를 올렸는데 화면이 안 바뀌는 사고 발생. 원인은 `git pull`이 실패하는데 스크립트가 멈추지 않고 계속 진행해서, 옛날 코드로 빌드 번호만 새로 찍혀 "성공한 것처럼 보이는" 문제. `git fetch + git reset --hard`로 해결.
- 다크모드에서 모든 글자가 안 보이는 문제. 원인은 CSS 투명도 애니메이션이 배경뿐만 아니라 글자까지 같이 투명하게 만드는 문제. `::before` 가상 요소로 분리해서 해결.

**이 날의 교훈:** "자동 배포에서 `git pull`은 절대 쓰면 안 된다. `git fetch + git reset --hard`만 써야 한다."

## 10. 핵심 특징

### 1. 자동 업무 배분
대표가 "삼성전자 주가 분석해줘"라고 하면, 비서실장이 자동으로 CIO(투자분석처장)에게 보내고, CIO가 4명 전문가에게 병렬로 배분합니다. 사람이 하나하나 지시할 필요 없습니다.

### 2. 자율 딥워크
에이전트가 1번만 답하는 게 아닙니다. 계획 -> 조사 -> 초안 -> 정리 -> 보고까지 **자율적으로 여러 단계를 수행**합니다. 마치 실제 직원이 보고서를 쓰듯이.

### 3. 실시간 대시보드
29명의 에이전트가 지금 뭘 하고 있는지, 비용이 얼마나 들었는지, 작업이 어디까지 진행됐는지를 웹 브라우저에서 실시간으로 봅니다.

### 4. 에이전트 성격 편집 (Soul)
각 에이전트의 성격, 말투, 전문 분야를 자유롭게 설정할 수 있습니다. "너는 침착하고 체계적인 비서실장이다" 같은 프롬프트를 웹에서 바로 수정 가능.

### 5. 가상 사무실
에이전트들이 부서별로 배치된 가상 사무실을 볼 수 있습니다. 각 에이전트의 상태(활동/대기/비활성)가 실시간으로 표시됩니다.

### 6. 노션 자동 보고 (계획 중)
